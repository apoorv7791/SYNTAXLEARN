Basic Syntax and Structure

C++ programs follow a specific structure and syntax. Understanding these fundamentals is essential for writing correct code.

Program Structure

A basic C++ program consists of:

#include <iostream>
using namespace std;

int main() {
    // Your code goes here
    cout << "Hello, World!" << endl;
    return 0;
}

Let's break down the components:

  • #include <iostream>: Includes the input/output stream library
  • using namespace std;: Allows us to use elements from the standard namespace without prefixing
  • int main(): The main function, where program execution begins
  • return 0;: Indicates successful program termination

Comments

Comments are used to explain code and are ignored by the compiler:

// This is a single-line comment

/* This is a
   multi-line comment */

Statements and Semicolons

In C++, statements are terminated with semicolons:

int x = 5;  // Declaration statement
cout << x;  // Output statement
x = x + 1;  // Assignment statement

Blocks of Code

Blocks are defined using curly braces and group multiple statements:

if (x > 0) {
    cout << "Positive";
    x = 0;
}

Key points to remember:

  • Every C++ program must have a main() function
  • Statements end with semicolons
  • Code blocks are enclosed in curly braces {}
  • C++ is case-sensitive (e.g., 'Variable' and 'variable' are different)
  • Whitespace (spaces, tabs, newlines) is generally ignored by the compiler
