Functions Kya Hote Hai? ü§î

Bro, functions basically code ke reusable blocks hote hai. Matlab ek baar likho, hazaar baar use karo! Functions se code organize hota hai, readable banta hai, aur duplication kam hoti hai. PHP mein bahut saare built-in functions hai, aur tum apne khud ke bhi bana sakte ho.

Functions Kaise Define Karte Hai? üìù

PHP mein function define karne ke liye, function keyword ke baad function ka naam aur parentheses likhte hai:

<?php
// Basic function definition
function sayHello() {
    echo "Hello, World!";
}

// Function ko call karo
sayHello();  // Output: Hello, World!
?>

Function Parameters üì¶

Bro, functions parameters (ya arguments) accept kar sakte hai, jo function ko pass kiye gaye values hote hai:

<?php
// Parameter wala function
function greet($name) {
    echo "Hello, $name!";
}

greet("Rahul");  // Output: Hello, Rahul!
greet("Priya");  // Output: Hello, Priya!

// Multiple parameters wala function
function add($a, $b) {
    echo "$a + $b = " . ($a + $b);
}

add(5, 3);  // Output: 5 + 3 = 8
?>

Default Parameter Values üéØ

Bro, tum parameters ke liye default values specify kar sakte ho, jo tab use hoti hai jab koi value pass nahi ki jati:

<?php
// Default parameter values wala function
function greetWithTitle($name, $title = "Mr.") {
    echo "Hello, $title $name!";
}

greetWithTitle("Sharma");         // Output: Hello, Mr. Sharma!
greetWithTitle("Patel", "Dr.");   // Output: Hello, Dr. Patel!

// Default parameters required parameters ke baad hi define karne chahiye
function setHeight($minHeight, $maxHeight = 100, $defaultHeight = 50) {
    echo "Min: $minHeight, Max: $maxHeight, Default: $defaultHeight";
}

setHeight(30);  // Output: Min: 30, Max: 100, Default: 50
setHeight(30, 80);  // Output: Min: 30, Max: 80, Default: 50
setHeight(30, 80, 60);  // Output: Min: 30, Max: 80, Default: 60
?>

Return Values üîô

Bro, functions return statement ke through values return kar sakte hai. Jaise hi return statement execute hota hai, function turant terminate ho jata hai:

<?php
// Value return karne wala function
function square($number) {
    return $number * $number;
}

$result = square(4);
echo "4 ka square hai $result";  // Output: 4 ka square hai 16

// Multiple return statements wala function
function getAbsoluteValue($number) {
    if ($number >= 0) {
        return $number;
    } else {
        return -$number;
    }
}

echo getAbsoluteValue(5);   // Output: 5
echo getAbsoluteValue(-5);  // Output: 5

// Array ke through multiple values return karna
function getMinMax($numbers) {
    return [
        'min' => min($numbers),
        'max' => max($numbers)
    ];
}

$result = getMinMax([3, 1, 7, 5]);
echo "Min: " . $result['min'] . ", Max: " . $result['max'];  // Output: Min: 1, Max: 7
?>

Variable Scope üîç

Bro, function ke andar declare kiye gaye variables ka local scope hota hai aur sirf usi function ke andar accessible hote hai. Function ke andar global variables access karne ke liye, global keyword use karo:

<?php
// Global variable
$message = "Hello, World!";

function displayMessage() {
    // Ye error dega kyunki $message yahan accessible nahi hai
    // echo $message;
    
    // Global variable access karne ke liye, global keyword use karo
    global $message;
    echo $message;
}

displayMessage();  // Output: Hello, World!

// Alternative tarika: $GLOBALS array use karo
function displayMessageAlternative() {
    echo $GLOBALS['message'];
}

displayMessageAlternative();  // Output: Hello, World!
?>

Static Variables üß†

Bro, static variables function calls ke beech mein apni value retain karte hai:

<?php
function counter() {
    static $count = 0;  // Sirf ek baar initialize hota hai
    $count++;
    echo "Count: $count<br>";
}

counter();  // Output: Count: 1
counter();  // Output: Count: 2
counter();  // Output: Count: 3

// Non-static variable ke saath compare karo
function counterNonStatic() {
    $count = 0;  // Har baar function call hone par initialize hota hai
    $count++;
    echo "Non-static count: $count<br>";
}

counterNonStatic();  // Output: Non-static count: 1
counterNonStatic();  // Output: Non-static count: 1
counterNonStatic();  // Output: Non-static count: 1
?>

Variable Functions üîÑ

Bro, PHP variable functions support karta hai, matlab agar variable name ke baad parentheses lagaye jate hai, to PHP us variable ke value ke naam ka function dhundhta hai aur use execute karta hai:

<?php
function sayHello() {
    echo "Hello!";
}

function sayGoodbye() {
    echo "Goodbye!";
}

$functionName = "sayHello";
$functionName();  // Output: Hello!

$functionName = "sayGoodbye";
$functionName();  // Output: Goodbye!

// Ye callbacks ke liye useful hai
function callFunction($callback) {
    $callback();
}

callFunction("sayHello");  // Output: Hello!
?>

Anonymous Functions (Closures) üïµÔ∏è

Bro, anonymous functions, jinhe closures bhi kehte hai, bina naam specify kiye functions create karne ki permission dete hai:

<?php
// Variable ko assign kiya gaya anonymous function
$greet = function($name) {
    echo "Hello, $name!";
};

$greet("Rahul");  // Output: Hello, Rahul!

// Argument ke roop mein anonymous function
$numbers = [1, 2, 3, 4, 5];
$squaredNumbers = array_map(function($n) {
    return $n * $n;
}, $numbers);

print_r($squaredNumbers);  // Output: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

// Closures 'use' keyword ke through parent scope se variables use kar sakte hai
$message = "Hello";
$greet = function($name) use ($message) {
    echo "$message, $name!";
};

$greet("Rahul");  // Output: Hello, Rahul!
?>

Arrow Functions (PHP 7.4+) üèπ

Bro, arrow functions simple anonymous functions ke liye ek more concise syntax provide karte hai:

<?php
// Traditional anonymous function
$numbers = [1, 2, 3, 4, 5];
$squaredNumbers = array_map(function($n) {
    return $n * $n;
}, $numbers);

// Equivalent arrow function
$squaredNumbers = array_map(fn($n) => $n * $n, $numbers);

print_r($squaredNumbers);  // Output: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

// Arrow functions automatically parent scope se variables capture karte hai
$multiplier = 2;
$multipliedNumbers = array_map(fn($n) => $n * $multiplier, $numbers);

print_r($multipliedNumbers);  // Output: Array ( [0] => 2 [1] => 4 [2] => 6 [3] => 8 [4] => 10 )
?>

Type Declarations üìã

Bro, PHP 7+ function parameters aur return values ke liye type declarations support karta hai:

<?php
// Parameters ke liye type declarations
function add(int $a, int $b) {
    return $a + $b;
}

echo add(5, 3);  // Output: 8
// echo add("5", "3");  // PHP 5 mein work karta hai, lekin PHP 7+ mein strict typing enabled hone par error dega

// Return type declarations
function multiply(int $a, int $b): int {
    return $a * $b;
}

echo multiply(5, 3);  // Output: 15

// Nullable types (PHP 7.1+)
function getUser(int $id): ?array {
    // User data ya null return karo agar user na mile
    if ($id > 0) {
        return ['id' => $id, 'name' => 'Rahul'];
    } else {
        return null;
    }
}

var_dump(getUser(1));  // Output: array(2) { ["id"]=> int(1) ["name"]=> string(5) "Rahul" }
var_dump(getUser(-1)); // Output: NULL

// Union types (PHP 8.0+)
function process(int|float $number): int|float {
    return $number * 2;
}

echo process(5);    // Output: 10
echo process(2.5);  // Output: 5
?>

Pro Tip: Bro, strict typing enable karne ke liye, apne PHP file ke beginning mein declare(strict_types=1); add karo. Isse PHP type mismatches hone par errors throw karega.

Variadic Functions üî¢

Bro, variadic functions variable number of arguments accept kar sakte hai. ... operator arguments ko array mein collect karne ke liye use hota hai:

<?php
// Variadic function
function sum(...$numbers) {
    return array_sum($numbers);
}

echo sum(1, 2, 3, 4, 5);  // Output: 15

// Regular parameters ko variadic parameters ke saath combine karna
function sumWithMultiplier($multiplier, ...$numbers) {
    return $multiplier * array_sum($numbers);
}

echo sumWithMultiplier(2, 1, 2, 3, 4, 5);  // Output: 30 (2 * (1+2+3+4+5))

// Arrays ko arguments mein unpack karna
$numbers = [1, 2, 3, 4, 5];
echo sum(...$numbers);  // Output: 15
?>

Recursive Functions üîÑ

Bro, recursive function wo function hota hai jo khud ko call karta hai:

<?php
// Factorial calculate karne ke liye recursive function
function factorial($n) {
    if ($n <= 1) {
        return 1;
    } else {
        return $n * factorial($n - 1);
    }
}

echo "5! = " . factorial(5);  // Output: 5! = 120 (5*4*3*2*1)

// Fibonacci sequence generate karne ke liye recursive function
function fibonacci($n) {
    if ($n <= 1) {
        return $n;
    } else {
        return fibonacci($n - 1) + fibonacci($n - 2);
    }
}

echo "Fibonacci(10) = " . fibonacci(10);  // Output: Fibonacci(10) = 55
?>

Warning Bro: Recursive functions ke saath careful raho kyunki agar proper base case na ho ya bahut deep recursion ho to stack overflow errors ho sakte hai.

Callback Functions üìû

Bro, callback functions wo functions hote hai jo dusre functions ko arguments ke roop mein pass kiye jate hai:

<?php
// Callback accept karne wala function
function processNumbers($numbers, $callback) {
    $result = [];
    foreach ($numbers as $number) {
        $result[] = $callback($number);
    }
    return $result;
}

// Callback function define karo
function square($n) {
    return $n * $n;
}

$numbers = [1, 2, 3, 4, 5];
$squaredNumbers = processNumbers($numbers, 'square');
print_r($squaredNumbers);  // Output: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

// Anonymous function ko callback ke roop mein use karna
$doubledNumbers = processNumbers($numbers, function($n) {
    return $n * 2;
});
print_r($doubledNumbers);  // Output: Array ( [0] => 2 [1] => 4 [2] => 6 [3] => 8 [4] => 10 )

// Built-in PHP functions jo callbacks accept karte hai
$evenNumbers = array_filter($numbers, function($n) {
    return $n % 2 == 0;
});
print_r($evenNumbers);  // Output: Array ( [1] => 2 [3] => 4 )
?>

Built-in Functions üõ†Ô∏è

Bro, PHP mein various purposes ke liye bahut saare built-in functions hai. Yahan kuch commonly used functions hai:

String Functions

<?php
$str = "Hello, World!";

echo strlen($str);           // Output: 13 (string length)
echo strtoupper($str);       // Output: HELLO, WORLD!
echo strtolower($str);       // Output: hello, world!
echo str_replace("World", "PHP", $str);  // Output: Hello, PHP!
echo substr($str, 0, 5);     // Output: Hello (substring)
?>

Array Functions

<?php
$fruits = ["apple", "banana", "cherry"];

echo count($fruits);         // Output: 3 (array size)
sort($fruits);               // Array sort karo
print_r($fruits);            // Output: Array ( [0] => apple [1] => banana [2] => cherry )
$reversed = array_reverse($fruits);  // Array reverse karo
print_r($reversed);          // Output: Array ( [0] => cherry [1] => banana [2] => apple )
echo implode(", ", $fruits); // Output: apple, banana, cherry (array elements ko join karo)
?>

Mathematical Functions

<?php
echo abs(-5);                // Output: 5 (absolute value)
echo round(3.7);             // Output: 4 (nearest integer tak round karo)
echo floor(3.7);             // Output: 3 (neeche round karo)
echo ceil(3.2);              // Output: 4 (upar round karo)
echo rand(1, 10);            // Output: 1 aur 10 ke beech random number
echo max(5, 10, 15);         // Output: 15 (maximum value)
echo min(5, 10, 15);         // Output: 5 (minimum value)
?>

Date and Time Functions

<?php
echo time();                 // Output: Current Unix timestamp
echo date("Y-m-d H:i:s");    // Output: Current date and time (e.g., 2025-04-14 19:30:00)
echo date("F j, Y, g:i a");  // Output: Formatted date (e.g., April 14, 2025, 7:30 pm)
echo strtotime("+1 week");   // Output: Abhi se 1 week baad ka Unix timestamp
echo date("Y-m-d", strtotime("+1 month"));  // Output: Abhi se 1 month baad ki date
?>

Function Best Practices üíØ

  • Single Responsibility: Bro, har function ka ek hi, well-defined purpose hona chahiye.
  • Meaningful Names: Descriptive names choose karo jo bataye ki function kya karta hai.
  • Keep Functions Short: Functions concise aur specific task par focused hone chahiye.
  • Use Type Declarations: Parameter aur return types specify karo taki code more robust bane.
  • Document Your Functions: Comments ya PHPDoc blocks use karke describe karo ki function kya karta hai, uske parameters, aur return values kya hai.
  • Return Values: Return values ke saath consistent raho. Agar function ek case mein value return karta hai, to sabhi cases mein value return karna chahiye.
  • Error Handling: Function ke andar potential errors handle karo ya errors report karne ke liye exceptions use karo.

Example: Complete Function Documentation üìö

<?php
/**
 * Rectangle ka area calculate karo.
 *
 * @param float $length Rectangle ki length.
 * @param float $width Rectangle ki width.
 * @return float Rectangle ka area.
 * @throws InvalidArgumentException Agar length ya width negative hai.
 */
function calculateRectangleArea(float $length, float $width): float {
    if ($length < 0 || $width < 0) {
        throw new InvalidArgumentException("Length aur width positive numbers hone chahiye.");
    }
    return $length * $width;
}

try {
    echo calculateRectangleArea(5, 3);  // Output: 15
    echo calculateRectangleArea(-5, 3); // Exception throw karega
} catch (InvalidArgumentException $e) {
    echo "Error: " . $e->getMessage();  // Output: Error: Length aur width positive numbers hone chahiye.
}
?>
