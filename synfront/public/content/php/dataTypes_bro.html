Variables Kya Hote Hai? ü§î

Bro, PHP mein variables data store karne ke liye use hote hai. PHP loosely typed language hai, matlab variable banate time data type declare karne ki zaroorat nahi hai. Kuch bhi store kar sakte ho - string, number, kuch bhi!

<?php
// Variables declare aur assign kaise karte hai
$name = "Rahul";
$age = 25;
$height = 1.75;
$isStudent = true;

// Variables ko output kaise karte hai
echo "Name: " . $name . "<br>";
echo "Age: " . $age . "<br>";
echo "Height: " . $height . " meters<br>";
echo "Is Student: " . ($isStudent ? "Haan" : "Nahi");
?>

Output:
Name: Rahul
Age: 25
Height: 1.75 meters
Is Student: Haan

Data Type Kaise Check Kare? üßê

Bro, PHP mein data type check karne ke liye kai functions hai:

<?php
$name = "Rahul";
$age = 25;
$height = 1.75;

echo "Variable \$isStudent ka type hai " . gettype($isStudent) . "<br>";
echo "Variable \$colors ka type hai " . gettype($colors) . "<br>";
echo "Variable \$person ka type hai " . gettype($person) . "<br>";

// Type-specific checking functions
echo "Kya \$name string hai? " . (is_string($name) ? "Haan" : "Nahi") . "<br>";
echo "Kya \$age integer hai? " . (is_int($age) ? "Haan" : "Nahi") . "<br>";
echo "Kya \$height float hai? " . (is_float($height) ? "Haan" : "Nahi") . "<br>";
echo "Kya \$isStudent boolean hai? " . (is_bool($isStudent) ? "Haan" : "Nahi") . "<br>";
echo "Kya \$colors array hai? " . (is_array($colors) ? "Haan" : "Nahi") . "<br>";
echo "Kya \$person null hai? " . (is_null($person) ? "Haan" : "Nahi") . "<br>";
?>

Variable $height ka type hai double
Variable $isStudent ka type hai boolean
Variable $colors ka type hai array
Variable $person ka type hai NULL
Kya $name string hai? Haan
Kya $age integer hai? Haan
Kya $height float hai? Haan
Kya $isStudent boolean hai? Haan
Kya $colors array hai? Haan
Kya $person null hai? Haan

Type Casting - Data Type Convert Karna üîÑ

Bro, PHP mein ek data type se dusre mein convert karne ke liye type casting use karte hai:

<?php
// String to integer
$strNumber = "123";
$intNumber = (int)$strNumber;
echo "String \"$strNumber\" ko integer mein convert kiya: $intNumber<br>";

// Integer to string
$num = 456;
$strNum = (string)$num;
echo "Integer $num ko string mein convert kiya: \"$strNum\"<br>";

// Float to integer (decimal part hat jata hai)
$floatNum = 7.89;
$intNum = (int)$floatNum;
echo "Float $floatNum ko integer mein convert kiya: $intNum<br>";

// Boolean to integer
$bool = true;
$intBool = (int)$bool;
echo "Boolean true ko integer mein convert kiya: $intBool<br>";

// String to array
$str = "Hello";
$arr = (array)$str;
echo "String \"$str\" ko array mein convert kiya: ";
print_r($arr);
echo "<br>";
?>

Output:
String "123" ko integer mein convert kiya: 123
Integer 456 ko string mein convert kiya: "456"
Float 7.89 ko integer mein convert kiya: 7
Boolean true ko integer mein convert kiya: 1
String "Hello" ko array mein convert kiya: Array ( [0] => Hello )

Numeric Data Types üî¢

Bro, PHP mein do numeric data types hai: integers aur floating-point numbers (floats).

Integers

Integer matlab bina decimal point wala number. Positive, negative, ya zero ho sakta hai:

<?php
// Decimal (base 10)
$decimal = 42;

// Hexadecimal (base 16)
$hex = 0x2A;  // 42 in decimal

// Octal (base 8)
$octal = 052;  // 42 in decimal

// Binary (base 2)
$binary = 0b101010;  // 42 in decimal

echo "Decimal: $decimal<br>";
echo "Hexadecimal: $hex<br>";
echo "Octal: $octal<br>";
echo "Binary: $binary<br>";

// Integer size
echo "Maximum integer value: " . PHP_INT_MAX . "<br>";
echo "Minimum integer value: " . PHP_INT_MIN . "<br>";
echo "Integer size: " . PHP_INT_SIZE . " bytes<br>";
?>

Floats

Bro, float (ya double) matlab decimal point wala number ya exponential form mein number:

<?php
// Regular float
$float1 = 3.14;

// Scientific notation
$float2 = 3.14e2;  // 3.14 * 10^2 = 314

echo "Float 1: $float1<br>";
echo "Float 2: $float2<br>";

// Float precision issues
$a = 0.1;
$b = 0.2;
$c = $a + $b;
echo "0.1 + 0.2 = $c<br>";  // Exactly 0.3 nahi hoga floating-point precision ki wajah se

// Floats ko safely compare karna
$epsilon = 0.00001;
echo "Kya 0.1 + 0.2 aur 0.3 equal hai? " . 
    (abs($c - 0.3) < $epsilon ? "Haan" : "Nahi") . "<br>";
?>

Pro Tip: Bro, floating-point numbers binary mein represent hote hai, isliye kuch decimal values exactly represent nahi ho sakte. Floats ke saath kaam karte time hamesha appropriate comparison methods use karo.

Boolean Data Type ‚úÖ

Bro, boolean matlab true ya false value:

<?php
$isTrue = true;
$isFalse = false;

echo "Is true: " . ($isTrue ? "Haan" : "Nahi") . "<br>";
echo "Is false: " . ($isFalse ? "Haan" : "Nahi") . "<br>";

// Ye values false ke tarah interpret hoti hai
echo "Empty string is false: " . ('' ? "Nahi" : "Haan") . "<br>";
echo "0 is false: " . (0 ? "Nahi" : "Haan") . "<br>";
echo "'0' is false: " . ('0' ? "Nahi" : "Haan") . "<br>";
echo "null is false: " . (null ? "Nahi" : "Haan") . "<br>";
echo "Empty array is false: " . ([] ? "Nahi" : "Haan") . "<br>";

// Ye values true ke tarah interpret hoti hai
echo "Non-empty string is true: " . ('hello' ? "Haan" : "Nahi") . "<br>";
echo "Non-zero number is true: " . (42 ? "Haan" : "Nahi") . "<br>";
echo "Non-empty array is true: " . ([1, 2, 3] ? "Haan" : "Nahi") . "<br>";
?>

Array Data Type üìö

Bro, array ek data structure hai jo ek variable mein multiple values store kar sakta hai. PHP mein indexed aur associative dono arrays support karta hai:

<?php
// Indexed array
$fruits = ["Apple", "Banana", "Cherry"];
echo "First fruit: " . $fruits[0] . "<br>";
echo "Second fruit: " . $fruits[1] . "<br>";
echo "Third fruit: " . $fruits[2] . "<br>";

// Associative array
$person = [
    "name" => "Rahul",
    "age" => 25,
    "city" => "Mumbai"
];
echo "Name: " . $person["name"] . "<br>";
echo "Age: " . $person["age"] . "<br>";
echo "City: " . $person["city"] . "<br>";

// Multidimensional array
$employees = [
    ["Rahul", "Sharma", "Manager"],
    ["Priya", "Patel", "Developer"],
    ["Amit", "Kumar", "Designer"]
];
echo "First employee: " . $employees[0][0] . " " . $employees[0][1] . 
    " (" . $employees[0][2] . ")<br>";

// Array functions
echo "Number of fruits: " . count($fruits) . "<br>";
sort($fruits);
echo "Sorted fruits: ";
print_r($fruits);
echo "<br>";

// Check karo ki key exist karti hai ya nahi
echo "Kya 'age' key exist karti hai? " . (array_key_exists("age", $person) ? "Haan" : "Nahi") . "<br>";

// Check karo ki value exist karti hai ya nahi
echo "Kya 'Banana' fruits mein hai? " . (in_array("Banana", $fruits) ? "Haan" : "Nahi") . "<br>";
?>

Object Data Type üß©

Bro, objects classes ke instances hote hai jinme properties aur methods ho sakte hai:

<?php
// Class define karo
class Person {
    // Properties
    public $name;
    public $age;
    
    // Constructor
    function __construct($name, $age) {
        $this->name = $name;
        $this->age = $age;
    }
    
    // Method
    function introduce() {
        return "Hi, main " . $this->name . " hun aur meri age " . $this->age . " hai.";
    }
}

// Object create karo
$person = new Person("Rahul", 25);

// Properties access karo
echo "Name: " . $person->name . "<br>";
echo "Age: " . $person->age . "<br>";

// Method call karo
echo "Introduction: " . $person->introduce() . "<br>";
?>

Variable Scope üîç

Bro, variable ka scope determine karta hai ki use kahan access kiya ja sakta hai. PHP mein teen variable scopes hai:

Local Scope

Function ke andar declare kiye gaye variables local scope rakhte hai aur sirf us function ke andar access kiye ja sakte hai:

<?php
function test() {
    $localVar = "Main local hun";
    echo "Function ke andar: $localVar<br>";
}

test();
// Ye error cause karega:
// echo "Function ke bahar: $localVar<br>";
?>

Global Scope

Function ke bahar declare kiye gaye variables global scope rakhte hai lekin by default function ke andar accessible nahi hote:

<?php
$globalVar = "Main global hun";

function testGlobal() {
    // Ye kaam nahi karega:
    // echo "Function ke andar: $globalVar<br>";
    
    // Global variables access karne ke liye, global keyword use karo:
    global $globalVar;
    echo "Function ke andar with global keyword: $globalVar<br>";
    
    // Ya fir $GLOBALS array use karo:
    echo "Function ke andar with GLOBALS array: " . $GLOBALS['globalVar'] . "<br>";
}

echo "Function ke bahar: $globalVar<br>";
testGlobal();
?>

Static Scope

Static variables function calls ke beech apni value retain karte hai:

<?php
function counter() {
    static $count = 0;
    $count++;
    echo "Count: $count<br>";
}

counter();  // Output: Count: 1
counter();  // Output: Count: 2
counter();  // Output: Count: 3
?>

Constants üóø

Bro, constants aise identifiers hai jo simple values store karte hai aur script execution ke dauraan change nahi ho sakte:

<?php

// Constants define karne ka ek aur tarika (PHP 5.3+)

echo "App name: " . APP_NAME . "<br>";
echo "App version: " . APP_VERSION . "<br>";

// Predefined constants

echo "Operating system: " . PHP_OS . "<br>";
?>

Pro Tip: Bro, constants use karo aise values ke liye jo application mein change nahi honi chahiye, jaise configuration settings, database connection details, ya mathematical constants.
