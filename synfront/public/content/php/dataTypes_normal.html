PHP Data Types and Variables

Variables in PHP

In PHP, variables are used to store data values. Unlike many other programming languages, PHP is loosely typed, which means you don't need to declare the data type of a variable when you create it.

<?php
// Variable declaration and assignment
$name = "John";
$age = 30;
$height = 1.85;
$isStudent = true;

// Output variables
echo "Name: " . $name . "<br>";
echo "Age: " . $age . "<br>";
echo "Height: " . $height . " meters<br>";
echo "Is Student: " . ($isStudent ? "Yes" : "No");
?>

Output:
Name: John
Age: 30
Height: 1.85 meters
Is Student: Yes

PHP Data Types

PHP supports the following data types:

Data Type Description Example
String A sequence of characters $name = "John";
Integer A non-decimal number $age = 30;
Float A decimal number $height = 1.85;
Boolean True or false $isStudent = true;
Array Stores multiple values in a single variable $colors = ["red", "green", "blue"];
Object Instances of classes $person = new Person();
NULL A variable with no value $result = null;
Resource Special variable holding a reference to an external resource $file = fopen("file.txt", "r");

Checking Data Types

PHP provides several functions to check the data type of a variable:

<?php
$name = "John";
$age = 30;
$height = 1.85;
$isStudent = true;
$colors = ["red", "green", "blue"];
$person = null;

// Check data types
echo "Variable \$name is " . gettype($name) . "<br>";
echo "Variable \$age is " . gettype($age) . "<br>";
echo "Variable \$height is " . gettype($height) . "<br>";
echo "Variable \$isStudent is " . gettype($isStudent) . "<br>";
echo "Variable \$colors is " . gettype($colors) . "<br>";
echo "Variable \$person is " . gettype($person) . "<br>";

// Type-specific checking functions
echo "Is \$name a string? " . (is_string($name) ? "Yes" : "No") . "<br>";
echo "Is \$age an integer? " . (is_int($age) ? "Yes" : "No") . "<br>";
echo "Is \$height a float? " . (is_float($height) ? "Yes" : "No") . "<br>";
echo "Is \$isStudent a boolean? " . (is_bool($isStudent) ? "Yes" : "No") . "<br>";
echo "Is \$colors an array? " . (is_array($colors) ? "Yes" : "No") . "<br>";
echo "Is \$person null? " . (is_null($person) ? "Yes" : "No") . "<br>";
?>

Output:
Variable $name is string
Variable $age is integer
Variable $height is double
Variable $isStudent is boolean
Variable $colors is array
Variable $person is NULL
Is $name a string? Yes
Is $age an integer? Yes
Is $height a float? Yes
Is $isStudent a boolean? Yes
Is $colors an array? Yes
Is $person null? Yes

Type Casting

PHP allows you to convert a variable from one data type to another using type casting:

<?php
// String to integer
$strNumber = "123";
$intNumber = (int)$strNumber;
echo "String \"$strNumber\" cast to integer: $intNumber<br>";

// Integer to string
$num = 456;
$strNum = (string)$num;
echo "Integer $num cast to string: \"$strNum\"<br>";

// Float to integer (truncates the decimal part)
$floatNum = 7.89;
$intNum = (int)$floatNum;
echo "Float $floatNum cast to integer: $intNum<br>";

// Boolean to integer
$bool = true;
$intBool = (int)$bool;
echo "Boolean true cast to integer: $intBool<br>";

// String to array
$str = "Hello";
$arr = (array)$str;
echo "String \"$str\" cast to array: ";
print_r($arr);
echo "<br>";
?>

Output:
String "123" cast to integer: 123
Integer 456 cast to string: "456"
Float 7.89 cast to integer: 7
Boolean true cast to integer: 1
String "Hello" cast to array: Array ( [0] => Hello )

String Functions

PHP provides many built-in functions to manipulate strings:

<?php
$str = "Hello, World!";

// String length
echo "Length: " . strlen($str) . "<br>";

// Word count
echo "Word count: " . str_word_count($str) . "<br>";

// Reverse a string
echo "Reversed: " . strrev($str) . "<br>";

// Find position of a substring
echo "Position of 'World': " . strpos($str, "World") . "<br>";

// Replace part of a string
echo "Replaced: " . str_replace("World", "PHP", $str) . "<br>";

// Convert to uppercase/lowercase
echo "Uppercase: " . strtoupper($str) . "<br>";
echo "Lowercase: " . strtolower($str) . "<br>";

// Substring
echo "Substring: " . substr($str, 0, 5) . "<br>";
?>

Output:
Length: 13
Word count: 2
Reversed: !dlroW ,olleH
Position of 'World': 7
Replaced: Hello, PHP!
Uppercase: HELLO, WORLD!
Lowercase: hello, world!
Substring: Hello

Numeric Data Types

PHP supports two numeric data types: integers and floating-point numbers (floats).

Integers

An integer is a whole number without a decimal point. It can be positive, negative, or zero:

<?php
// Decimal (base 10)
$decimal = 42;

// Hexadecimal (base 16)
$hex = 0x2A;  // 42 in decimal

// Octal (base 8)
$octal = 052;  // 42 in decimal

// Binary (base 2)
$binary = 0b101010;  // 42 in decimal

echo "Decimal: $decimal<br>";
echo "Hexadecimal: $hex<br>";
echo "Octal: $octal<br>";
echo "Binary: $binary<br>";

// Integer size
echo "Maximum integer value: " . PHP_INT_MAX . "<br>";
echo "Minimum integer value: " . PHP_INT_MIN . "<br>";
echo "Integer size: " . PHP_INT_SIZE . " bytes<br>";
?>

Floats

A float (or double) is a number with a decimal point or a number in exponential form:

<?php
// Regular float
$float1 = 3.14;

// Scientific notation
$float2 = 3.14e2;  // 3.14 * 10^2 = 314

echo "Float 1: $float1<br>";
echo "Float 2: $float2<br>";

// Float precision issues
$a = 0.1;
$b = 0.2;
$c = $a + $b;
echo "0.1 + 0.2 = $c<br>";  // May not be exactly 0.3 due to floating-point precision

// Comparing floats safely
$epsilon = 0.00001;
echo "Are 0.1 + 0.2 and 0.3 equal? " . 
    (abs($c - 0.3) < $epsilon ? "Yes" : "No") . "<br>";
?>

Tip: Due to the way floating-point numbers are represented in binary, some decimal values cannot be represented exactly. Always use appropriate comparison methods when working with floats.

Boolean Data Type

A boolean represents a truth value: either true or false:

<?php
$isTrue = true;
$isFalse = false;

echo "Is true: " . ($isTrue ? "Yes" : "No") . "<br>";
echo "Is false: " . ($isFalse ? "Yes" : "No") . "<br>";

// Values interpreted as false
echo "Empty string is false: " . ('' ? "No" : "Yes") . "<br>";
echo "0 is false: " . (0 ? "No" : "Yes") . "<br>";
echo "'0' is false: " . ('0' ? "No" : "Yes") . "<br>";
echo "null is false: " . (null ? "No" : "Yes") . "<br>";
echo "Empty array is false: " . ([] ? "No" : "Yes") . "<br>";

// Values interpreted as true
echo "Non-empty string is true: " . ('hello' ? "Yes" : "No") . "<br>";
echo "Non-zero number is true: " . (42 ? "Yes" : "No") . "<br>";
echo "Non-empty array is true: " . ([1, 2, 3] ? "Yes" : "No") . "<br>";
?>

NULL Data Type

NULL represents a variable with no value:

<?php
$var = null;

echo "Is null: " . (is_null($var) ? "Yes" : "No") . "<br>";
echo "Is set: " . (isset($var) ? "Yes" : "No") . "<br>";

// Unsetting a variable
$name = "John";
echo "Name before unset: " . $name . "<br>";
unset($name);
echo "Is name set after unset: " . (isset($name) ? "Yes" : "No") . "<br>";
?>

Array Data Type

An array is a data structure that stores multiple values in a single variable. PHP supports both indexed and associative arrays:

<?php
// Indexed array
$fruits = ["Apple", "Banana", "Cherry"];
echo "First fruit: " . $fruits[0] . "<br>";
echo "Second fruit: " . $fruits[1] . "<br>";
echo "Third fruit: " . $fruits[2] . "<br>";

// Associative array
$person = [
    "name" => "John",
    "age" => 30,
    "city" => "New York"
];
echo "Name: " . $person["name"] . "<br>";
echo "Age: " . $person["age"] . "<br>";
echo "City: " . $person["city"] . "<br>";

// Multidimensional array
$employees = [
    ["John", "Doe", "Manager"],
    ["Jane", "Smith", "Developer"],
    ["Bob", "Johnson", "Designer"]
];
echo "First employee: " . $employees[0][0] . " " . $employees[0][1] . 
    " (" . $employees[0][2] . ")<br>";

// Array functions
echo "Number of fruits: " . count($fruits) . "<br>";
sort($fruits);
echo "Sorted fruits: ";
print_r($fruits);
echo "<br>";

// Check if key exists
echo "Does 'age' key exist? " . (array_key_exists("age", $person) ? "Yes" : "No") . "<br>";

// Check if value exists
echo "Is 'Banana' in fruits? " . (in_array("Banana", $fruits) ? "Yes" : "No") . "<br>";
?>

Object Data Type

Objects are instances of classes that can contain properties and methods:

<?php
// Define a class
class Person {
    // Properties
    public $name;
    public $age;
    
    // Constructor
    function __construct($name, $age) {
        $this->name = $name;
        $this->age = $age;
    }
    
    // Method
    function introduce() {
        return "Hi, I'm " . $this->name . " and I'm " . $this->age . " years old.";
    }
}

// Create an object
$person = new Person("John", 30);

// Access properties
echo "Name: " . $person->name . "<br>";
echo "Age: " . $person->age . "<br>";

// Call method
echo "Introduction: " . $person->introduce() . "<br>";
?>

Variable Scope

The scope of a variable determines where it can be accessed. PHP has three variable scopes:

Local Scope

Variables declared within a function have local scope and can only be accessed within that function:

<?php
function test() {
    $localVar = "I'm local";
    echo "Inside function: $localVar<br>";
}

test();
// This will cause an error:
// echo "Outside function: $localVar<br>";
?>

Global Scope

Variables declared outside a function have global scope but are not accessible inside functions by default:

<?php
$globalVar = "I'm global";

function testGlobal() {
    // This won't work:
    // echo "Inside function: $globalVar<br>";
    
    // To access global variables, use the global keyword:
    global $globalVar;
    echo "Inside function with global keyword: $globalVar<br>";
    
    // Or use the $GLOBALS array:
    echo "Inside function with GLOBALS array: " . $GLOBALS['globalVar'] . "<br>";
}

echo "Outside function: $globalVar<br>";
testGlobal();
?>

Static Scope

Static variables retain their value between function calls:

<?php
function counter() {
    static $count = 0;
    $count++;
    echo "Count: $count<br>";
}

counter();  // Outputs: Count: 1
counter();  // Outputs: Count: 2
counter();  // Outputs: Count: 3
?>

Constants

Constants are identifiers for simple values that cannot be changed during script execution:

<?php
// Define a constant
define("PI", 3.14159);

// Use the constant
echo "Value of PI: " . PI . "<br>";

// Another way to define constants (PHP 5.3+)
const APP_NAME = "My PHP App";
const APP_VERSION = "1.0.0";

echo "App name: " . APP_NAME . "<br>";
echo "App version: " . APP_VERSION . "<br>";

// Predefined constants
echo "PHP version: " . PHP_VERSION . "<br>";
echo "Operating system: " . PHP_OS . "<br>";
?>

Best Practice: Use constants for values that should not change throughout your application, such as configuration settings, database connection details, or mathematical constants.
