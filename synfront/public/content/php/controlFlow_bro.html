PHP Control Flow - Bro Mode 😎

Control Flow Kya Hota Hai? 🤔

Bro, control flow statements se tum apne code ke execution ko control kar sakte ho based on certain conditions. PHP mein ye control flow structures hai:

  • Conditional statements (if, else, elseif, switch) - Conditions check karne ke liye
  • Loops (for, while, do-while, foreach) - Code ko baar baar run karne ke liye
  • Jump statements (break, continue, return) - Flow ko change karne ke liye

Conditional Statements 🔀

If Statement

Bro, if statement tab code execute karta hai jab condition true hoti hai:

<?php
$age = 20;

if ($age >= 18) {
    echo "Tum adult ho.";
}
?>

Output:
Tum adult ho.

If-Else Statement

Bro, if-else statement ek code block execute karta hai jab condition true hoti hai, aur dusra block jab false hoti hai:

<?php
$age = 15;

if ($age >= 18) {
    echo "Tum adult ho.";
} else {
    echo "Tum minor ho.";
}
?>

Output:
Tum minor ho.

If-Elseif-Else Statement

Bro, if-elseif-else statement se tum multiple conditions check kar sakte ho:

<?php
$score = 85;

if ($score >= 90) {
    echo "Grade: A";
} elseif ($score >= 80) {
    echo "Grade: B";
} elseif ($score >= 70) {
    echo "Grade: C";
} elseif ($score >= 60) {
    echo "Grade: D";
} else {
    echo "Grade: F";
}
?>

Output:
Grade: B

Nested If Statements

Bro, tum ek if statement ke andar dusra if statement bhi rakh sakte ho:

<?php
$age = 25;
$hasLicense = true;

if ($age >= 18) {
    echo "Tum adult ho. ";
    
    if ($hasLicense) {
        echo "Tum drive kar sakte ho.";
    } else {
        echo "Tumhe drive karne ke liye license chahiye.";
    }
} else {
    echo "Tum minor ho. Tum drive nahi kar sakte.";
}
?>

Output:
Tum adult ho. Tum drive kar sakte ho.

Switch Statement

Bro, switch statement different conditions ke basis par different actions perform karne ke liye use hota hai:

<?php
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Work week ki shuruat hai.";
        break;
    case "Tuesday":
    case "Wednesday":
    case "Thursday":
        echo "Work day hai.";
        break;
    case "Friday":
        echo "Weekend almost aa gaya hai.";
        break;
    case "Saturday":
    case "Sunday":
        echo "Weekend hai bro!";
        break;
    default:
        echo "Invalid day.";
}
?>

Output:
Work week ki shuruat hai.

Important Tip: Bro, har case ke baad break statement lagana mat bhulna. Agar break nahi lagaya to PHP next cases ke statements bhi execute karega.

Ternary Operator

Bro, ternary operator if-else statement ka shorthand hai:

<?php
$age = 20;

// Syntax: condition ? true_wala_expression : false_wala_expression
$status = ($age >= 18) ? "adult" : "minor";
echo "Tum ek $status ho.";

// Nested ternary operators (readability ke liye dhyan se use karo)
$score = 85;
$grade = ($score >= 90) ? "A" : (($score >= 80) ? "B" : (($score >= 70) ? "C" : (($score >= 60) ? "D" : "F")));
echo "Grade: $grade";
?>

Output:
Tum ek adult ho.
Grade: B

Null Coalescing Operator

Bro, null coalescing operator (??) first operand return karta hai agar wo exist karta hai aur NULL nahi hai; warna second operand return karta hai:

<?php
// Assume $_GET['user'] set nahi hai
$username = $_GET['user'] ?? 'Guest';
echo "Hello, $username";

// Null coalescing operator ko chain karna
$username = $_GET['user'] ?? $_POST['user'] ?? 'Guest';
echo "Hello, $username";
?>

Output:
Hello, Guest
Hello, Guest

Loops - Code Ko Baar Baar Run Karna 🔄

Bro, loops se tum ek code block ko baar baar execute kar sakte ho.

For Loop

Bro, for loop tab use karo jab tumhe pehle se pata ho ki code kitni baar run karna hai:

<?php
// Syntax: for (initialization; condition; increment/decrement)
for ($i = 1; $i <= 5; $i++) {
    echo "Number: $i<br>";
}
?>

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

While Loop

Bro, while loop tab tak code execute karta hai jab tak condition true hai:

<?php
$i = 1;
while ($i <= 5) {
    echo "Number: $i<br>";
    $i++;
}
?>

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5

Do-While Loop

Bro, do-while loop while loop jaisa hai, lekin ye code block ko ek baar zaroor execute karega, phir condition check karega:

<?php
$i = 1;
do {
    echo "Number: $i<br>";
    $i++;
} while ($i <= 5);

// Even if condition initially false hai, code ek baar execute hoga
$j = 10;
do {
    echo "Ye ek baar print hoga even though j > 5 hai.";
} while ($j <= 5);
?>

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
Ye ek baar print hoga even though j > 5 hai.

Foreach Loop

Bro, foreach loop arrays ke through iterate karne ke liye use hota hai:

<?php
// Indexed array
$colors = ["red", "green", "blue"];

foreach ($colors as $color) {
    echo "Color: $color<br>";
}

// Associative array
$person = [
    "name" => "Rahul",
    "age" => 25,
    "city" => "Mumbai"
];

foreach ($person as $key => $value) {
    echo "$key: $value<br>";
}
?>

Output:
Color: red
Color: green
Color: blue
name: Rahul
age: 25
city: Mumbai

Jump Statements - Flow Ko Control Karna 🦘

Bro, jump statements se tum loop ya conditional statement ke flow ko alter kar sakte ho.

Break Statement

Bro, break statement loop ya switch statement ke execution ko terminate karta hai:

<?php
// For loop mein break
for ($i = 1; $i <= 10; $i++) {
    if ($i == 6) {
        break;  // Jab i equals 6 ho to loop se bahar nikal jao
    }
    echo "Number: $i<br>";
}

// Switch statement mein break
$fruit = "apple";
switch ($fruit) {
    case "apple":
        echo "Ye apple hai.";
        break;  // Switch statement se bahar nikal jao
    case "banana":
        echo "Ye banana hai.";
        break;
    default:
        echo "Ye kuch aur hai.";
}
?>

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
Ye apple hai.

Continue Statement

Bro, continue statement loop ke current iteration ko skip karke next iteration pe jump karta hai:

<?php
// Even numbers ko skip karo
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 == 0) {
        continue;  // Even numbers ke liye loop body ke baaki part ko skip karo
    }
    echo "Odd number: $i<br>";
}
?>

Output:
Odd number: 1
Odd number: 3
Odd number: 5
Odd number: 7
Odd number: 9

Return Statement

Bro, return statement function se bahar nikalne aur value return karne ke liye use hota hai:

<?php
function add($a, $b) {
    return $a + $b;  // Sum return karo aur function se bahar niklo
    echo "Ye line kabhi execute nahi hogi.";
}

$result = add(5, 3);
echo "Result: $result";
?>

Output:
Result: 8

Control Structures Ka Alternative Syntax 🔄

Bro, PHP mein control structures ke liye alternative syntax bhi hai, jo especially useful hai jab PHP ko HTML ke saath mix karte ho:

<?php
$user_logged_in = true;
?>

<!-- If-else ke liye alternative syntax -->
<?php if ($user_logged_in): ?>
    <h2>Welcome, User!</h2>
    <p>Tum logged in ho.</p>
<?php else: ?>
    <h2>Welcome, Guest!</h2>
    <p>Please log in karo.</p>
<?php endif; ?>

<!-- Foreach ke liye alternative syntax -->
<?php
$fruits = ["Apple", "Banana", "Cherry"];
?>
<ul>
    <?php foreach ($fruits as $fruit): ?>
        <li><?php echo $fruit; ?></li>
    <?php endforeach; ?>
</ul>

<!-- Switch ke liye alternative syntax -->
<?php
$day = "Monday";
switch ($day):
    case "Monday":
        echo "Aaj Monday hai.";
        break;
    case "Tuesday":
        echo "Aaj Tuesday hai.";
        break;
    default:
        echo "Aaj koi aur din hai.";
endswitch;
?>

Practical Examples 🛠️

Form Validation

Bro, form validation ke liye control flow ka use:

<?php
// Check karo ki form submit hua hai ya nahi
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Error array initialize karo
    $errors = [];
    
    // Name validate karo
    if (empty($_POST["name"])) {
        $errors[] = "Name required hai";
    } elseif (strlen($_POST["name"]) < 3) {
        $errors[] = "Name kam se kam 3 characters hona chahiye";
    }
    
    // Email validate karo
    if (empty($_POST["email"])) {
        $errors[] = "Email required hai";
    } elseif (!filter_var($_POST["email"], FILTER_VALIDATE_EMAIL)) {
        $errors[] = "Invalid email format";
    }
    
    // Errors ya success message display karo
    if (!empty($errors)) {
        echo "<ul>";
        foreach ($errors as $error) {
            echo "<li>$error</li>";
        }
        echo "</ul>";
    } else {
        echo "Form successfully submit ho gaya!";
    }
}
?>

<form method="post" action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]); ?>">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br><br>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br><br>
    
    <input type="submit" value="Submit">
</form>

Pagination

Bro, pagination ke liye control flow ka use:

<?php
// Sample data (real application mein, ye database se aayega)
$items = [];
for ($i = 1; $i <= 100; $i++) {
    $items[] = "Item $i";
}

// Pagination settings
$items_per_page = 10;
$total_items = count($items);
$total_pages = ceil($total_items / $items_per_page);

// Query string se current page get karo
$current_page = isset($_GET['page']) ? (int)$_GET['page'] : 1;

// Ensure current page valid hai
if ($current_page < 1) {
    $current_page = 1;
} elseif ($current_page > $total_pages) {
    $current_page = $total_pages;
}

// Current page ke items ke liye start index calculate karo
$start_index = ($current_page - 1) * $items_per_page;

// Current page ke items get karo
$current_items = array_slice($items, $start_index, $items_per_page);

// Items display karo
echo "<h3>Page $current_page of $total_pages</h3>";
echo "<ul>";
foreach ($current_items as $item) {
    echo "<li>$item</li>";
}
echo "</ul>";

// Pagination links display karo
echo "<div>";
if ($current_page > 1) {
    echo "<a href='?page=" . ($current_page - 1) . "'>Previous</a> ";
}

for ($i = 1; $i <= $total_pages; $i++) {
    if ($i == $current_page) {
        echo "<strong>$i</strong> ";
    } else {
        echo "<a href='?page=$i'>$i</a> ";
    }
}

if ($current_page < $total_pages) {
    echo "<a href='?page=" . ($current_page + 1) . "'>Next</a>";
}
echo "</div>";
?>

Best Practices 💯

  • Simple rakho: Bro, deeply nested control structures se bacho kyunki isse code read aur maintain karna mushkil ho jata hai.
  • Meaningful variable names use karo: Isse tumhare conditions zyada readable ho jate hai.
  • Multiple if-elseif ki jagah switch use karo: Jab ek hi variable ko multiple values ke against compare kar rahe ho, to switch statement zyada clear hota hai.
  • Loop conditions ke saath careful raho: Ensure karo ki tumhare loops eventually terminate honge to avoid infinite loops.
  • Alternative syntax use karo: Jab PHP ko HTML ke saath mix kar rahe ho, to control structures ke liye alternative syntax tumhare code ko zyada readable bana sakti hai.
  • Hamesha curly braces use karo: Even single-line blocks ke liye, curly braces use karna tumhare code ko more maintainable aur less error-prone banata hai.
