Apna Setup Ready Karo Bhai! 🚀

Bhai, C++ likhne ke liye pehle **setup set karna padega**! Nahi toh code likh ke bhi koi fayda nahi! 😆 Yeh do cheezein ready rakh:

  • **Visual Studio Code + C++ extension** (Yeh tera **coding adda** hai!🔥)
  • **g++ compiler** (Jo code ko computer ki bhasha me convert karega! 💻)

Jab yeh setup **set ho jaye**, tab **apna pehla C++ program likhke** try kar:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Bhai, yeh code **sirf ek kaam karega** → `"Hello, World!"` print karega. 😎 **Matlab dekhne me simple lag raha hoga, par yeh tera C++ ka pehla kadam hai!**

Thoda samajh le kya ho raha hai:

  • **#include <iostream>** → Bhai yeh zaroori hai, bina iske `cout` nahi chalega! Samajh le ek **toolbox** jisme input-output ke tools hain! ⚒️
  • **cout** → Matlab "Bro, yeh cheez print kar de!" Jo likhega woh console pe dikh jayega! 🖥️
  • **return 0;** → Matlab "Bhai, sab sahi chal gaya!" Agar iske bina chalaya toh error aayega! 😆

Yeh Cheezein Mat Bhoolna Bhai! 🧠

  • **Header files** hamesha include kar warna code ki waat lag jayegi! 😂
  • **main() function** bina program chalega nahi, iska dhyan rakh! 🚀
  • **cout output dene ke liye hai, cin input lene ke liye!** Isko samajh le warna confusion ho jayegi! 🤯
  • **Har statement ke end me semicolon ( ; ) zaroor daal!** Yeh mat bhoolna warna compiler tereko fatka dega! 🤦‍♂️😂

Bhai, ab tension mat le, bas **copy kar, VS Code me daal aur chala de!** Agar dikkat aaye toh tera bhai yahin hai! 😎🔥
