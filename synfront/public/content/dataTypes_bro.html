Data Types aur Variables - Coding ke Dabba! 📦

Bhai, C++ mein data types aur variables samajhna bahut zaroori hai! Yeh tere data ko store karne ke dabba hain! 🧰

Basic Data Types - Alag Alag Dabbe! 🎁

C++ mein kuch main data types hain, inko samajh le:

Data Type Kya Hai Yeh? Size (bytes) Example
int Poore numbers ke liye 4 int age = 25;
float Point wale numbers 4 float price = 19.99f;
double Bade point wale numbers 8 double pi = 3.14159265359;
char Ek character 1 char grade = 'A';
bool Haan ya Naa (true/false) 1 bool isActive = true;

Variable Declare aur Initialize Karna! 🚀

Variables ko use karne se pehle declare karna zaroori hai, bilkul phone number save karne jaisa:

// Sirf declare karna
int count;

// Declare + value set karna
int age = 25;
float salary = 50000.50;
char initial = 'J';
bool isEmployed = true;

Type Modifiers - Extra Features! 🔧

Type modifiers se data types ko customize kar sakte hain:

  • short: Chota int, kam memory leta hai! 🐜
  • long: Bada int ya double, zyada range ke liye! 🐘
  • signed: Positive aur negative dono values (default) ➕➖
  • unsigned: Sirf positive values ke liye ➕
short int smallNumber = 32767;  // Chota dabba
long int bigNumber = 2147483647;  // Bada dabba
unsigned int positiveOnly = 4294967295;  // No negative vibes!

Constants - Jo Kabhi Change Na Ho! 🔒

Constants woh variables hain jinki value kabhi change nahi hoti, ekdum pathar ki lakeer:

const double PI = 3.14159;  // Yeh value kabhi change nahi hogi
const int MAX_USERS = 100;  // Maximum users fix hai!

Type Conversion - Ek Type se Doosre Mein Convert! 🔄

C++ mein type conversion do tarah se hota hai:

// Implicit conversion - automatic
int x = 10;
double y = x;  // int se double mein convert ho gaya

// Explicit conversion - manually
double a = 3.14;
int b = (int)a;  // double se int mein convert, b = 3 ho jayega
int c = static_cast<int>(a);  // Modern C++ style casting

Yaad Rakhne Wali Baatein! 🧠

  • Variables ko hamesha initialize karo, warna random values aa sakti hain! 🎲
  • Sahi data type choose karo, memory bachao! 💾
  • Type conversion mein data loss ho sakta hai, dhyan rakho! ⚠️
  • Constants ka use karo jahan value fix ho! 🔒
  • Variable names case-sensitive hote hain, aur descriptive rakho! 📝
