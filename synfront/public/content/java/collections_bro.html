Collection Interfaces - Framework ke Building Blocks! 🧱

Collection - Sabka Baap! 👑

Collection interface hierarchy ka root hai. Yeh objects ke group ko represent karta hai, jinhe elements kehte hain.

List - Order Wala Collection! 📋

List ek ordered collection hai (sequence) jisme duplicate elements ho sakte hain. Index number se elements access kar sakte ho - bilkul array ki tarah!

Set - No Duplicates Wala Collection! 🚫

Set mein duplicate elements nahi ho sakte. Yeh mathematical set ki tarah work karta hai - har cheez unique!

Queue - Line System! 🧍‍♂️🧍‍♀️🧍‍♂️

Queue elements ko process karne se pehle hold karne ke liye designed hai. First-In-First-Out (FIFO) system - bilkul movie ticket line ki tarah!

Deque - Double Ended Queue! 🔄

Double-ended queue jisme dono ends se elements add aur remove kar sakte ho - dono taraf se entry-exit wala system!

Map - Key-Value Pairs! 🔑➡️📦

Map keys ko values se map karta hai. Duplicate keys nahi ho sakti; har key sirf ek value se map ho sakti hai - dictionary jaisa!

Common Implementations - Asli Collections! 🛠️

Interface Implementation Kya Khaas Hai?
List ArrayList Resizable array implementation. Random access fast hai par middle mein insertion/deletion slow hai. Array jaisa par size fixed nahi! 📏
LinkedList Doubly-linked list implementation. Insertions aur deletions fast hain par random access slow hai. Train ke dabbo ki tarah connected! 🚂
Vector Legacy synchronized implementation. ArrayList jaisa par thread-safe hai. Purana player hai market ka! 👴
Set HashSet Hash table implementation. Order guaranteed nahi hai par operations constant-time mein hote hain. Super fast lookup! ⚡
TreeSet Sorted set implementation using a tree. Elements sorted order mein rehte hain. Alphabetical order mein books ki tarah! 📚
Map HashMap Hash table implementation. Order guaranteed nahi hai par operations constant-time mein hote hain. Dictionary jaisa fast lookup! 📖
TreeMap Sorted map implementation using a tree. Keys sorted order mein rehti hain. Phone directory jaisa sorted! 📱
LinkedHashMap Hash table + linked list implementation. Insertion order maintain karta hai. History maintain karne wala HashMap! 🕰️
Queue PriorityQueue Priority heap implementation. Elements priority ke hisaab se order hote hain. VIP line system! 🌟
ArrayDeque Resizable array implementation. Queue aur stack dono ki tarah use kar sakte ho. 2-in-1 collection! 🔄

Collections Use Karna - Practical Examples! 💻

ArrayList Example - Sabse Popular List! 🌟

import java.util.ArrayList;
import java.util.List;

public class ArrayListExample {
    public static void main(String[] args) {
        // ArrayList banao
        List<String> fruits = new ArrayList<>();
        
        // Elements add karo
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        
        // Elements access karo
        System.out.println("Pehla fruit: " + fruits.get(0));  // Apple
        
        // Saare elements pe loop chalao
        System.out.println("Saare fruits:");
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
        
        // Check karo ki element exist karta hai ya nahi
        boolean containsBanana = fruits.contains("Banana");
        System.out.println("Banana hai kya? " + containsBanana);  // true
        
        // Element remove karo
        fruits.remove("Banana");
        
        // Size pata karo
        System.out.println("Fruits ki total count: " + fruits.size());  // 2
    }
}

HashSet Example - Duplicates Hata Do! 🚫

import java.util.HashSet;
import java.util.Set;

public class HashSetExample {
    public static void main(String[] args) {
        // HashSet banao
        Set<String> uniqueFruits = new HashSet<>();
        
        // Elements add karo
        uniqueFruits.add("Apple");
        uniqueFruits.add("Banana");
        uniqueFruits.add("Orange");
        uniqueFruits.add("Apple");  // Duplicate, add nahi hoga!
        
        // Saare elements pe loop chalao
        System.out.println("Saare unique fruits:");
        for (String fruit : uniqueFruits) {
            System.out.println(fruit);  // Apple, Orange, Banana (koi bhi order mein)
        }
        
        // Check karo ki element exist karta hai ya nahi
        boolean containsBanana = uniqueFruits.contains("Banana");
        System.out.println("Banana hai kya? " + containsBanana);  // true
        
        // Element remove karo
        uniqueFruits.remove("Banana");
        
        // Size pata karo
        System.out.println("Unique fruits ki total count: " + uniqueFruits.size());  // 2
    }
}

HashMap Example - Key-Value Pairs! 🔑

import java.util.HashMap;
import java.util.Map;

public class HashMapExample {
    public static void main(String[] args) {
        // HashMap banao
        Map<String, Double> fruitPrices = new HashMap<>();
        
        // Key-value pairs add karo
        fruitPrices.put("Apple", 1.99);
        fruitPrices.put("Banana", 0.59);
        fruitPrices.put("Orange", 1.29);
        
        // Key se value access karo
        Double applePrice = fruitPrices.get("Apple");
        System.out.println("Apple ki price: $" + applePrice);  // $1.99
        
        // Saare key-value pairs pe loop chalao
        System.out.println("Saare fruits ki prices:");
        for (Map.Entry<String, Double> entry : fruitPrices.entrySet()) {
            System.out.println(entry.getKey() + ": $" + entry.getValue());
        }
        
        // Check karo ki key exist karti hai ya nahi
        boolean containsBanana = fruitPrices.containsKey("Banana");
        System.out.println("Banana ki price available hai? " + containsBanana);  // true
        
        // Key-value pair remove karo
        fruitPrices.remove("Banana");
        
        // Size pata karo
        System.out.println("Price wale fruits ki total count: " + fruitPrices.size());  // 2
    }
}

Collections Utility Class - Swiss Army Knife! 🔧

Collections class static methods provide karta hai collections pe operate karne ke liye - ekdum multi-purpose tool ki tarah!

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class CollectionsUtilityExample {
    public static void main(String[] args) {
        List<Integer> numbers = new ArrayList<>();
        numbers.add(5);
        numbers.add(2);
        numbers.add(8);
        numbers.add(1);
        
        // List ko sort karo
        Collections.sort(numbers);
        System.out.println("Sorted numbers: " + numbers);  // [1, 2, 5, 8]
        
        // Binary search (sorted list pe)
        int index = Collections.binarySearch(numbers, 5);
        System.out.println("5 ka index: " + index);  // 2
        
        // Min aur max find karo
        int min = Collections.min(numbers);
        int max = Collections.max(numbers);
        System.out.println("Min: " + min + ", Max: " + max);  // Min: 1, Max: 8
        
        // List ko reverse karo
        Collections.reverse(numbers);
        System.out.println("Reversed: " + numbers);  // [8, 5, 2, 1]
        
        // List ko shuffle karo
        Collections.shuffle(numbers);
        System.out.println("Shuffled: " + numbers);  // Random order
        
        // Unmodifiable list banao
        List<Integer> unmodifiableList = Collections.unmodifiableList(numbers);
        // unmodifiableList.add(10);  // Yeh error throw karega - read-only list!
        
        // Synchronized list banao
        List<Integer> synchronizedList = Collections.synchronizedList(new ArrayList<>());
        // Ab list thread-safe hai - multiple threads ek saath access kar sakte hain!
    }
}

Sahi Collection Choose Karna - Konsa Use Karein? 🤔

Agar Chahiye... Use Karo...
Ordered collection with fast random access ArrayList - index se jaldi access karne ke liye! 📏
Fast insertions/deletions in the middle LinkedList - beech mein add/remove karne ke liye! 🚂
No duplicates and fast lookup HashSet - unique items aur fast search ke liye! 🔍
No duplicates and sorted elements TreeSet - sorted unique items ke liye! 📊
Key-value pairs with fast lookup HashMap - key se value fast find karne ke liye! 🔑
Key-value pairs with sorted keys TreeMap - sorted keys ke liye! 📑
FIFO (First-In-First-Out) behavior Queue (LinkedList ya ArrayDeque) - line system ke liye! 🧍‍♂️🧍‍♀️
LIFO (Last-In-First-Out) behavior Stack ya Deque (ArrayDeque) - plates stack karne jaisa! 🍽️
Elements ordered by priority PriorityQueue - VIP system ke liye! 🌟

Yaad Rakhne Wali Baatein! 🧠

  • Collections Framework ek unified architecture hai objects ke groups ke saath kaam karne ke liye - organized tarika! 🗂️
  • Apni requirement ke hisaab se sahi collection choose karo (ordering, duplicates, access patterns) - right tool for the job! 🛠️
  • ArrayList random access ke liye fast hai, LinkedList insertions aur deletions ke liye fast hai - use case ke hisaab se choose karo! ⚡
  • Sets mein duplicates nahi ho sakte, Lists mein ho sakte hain - requirements ke hisaab se choose karo! 🚫
  • Maps key-value pairs store karte hain aur key se fast lookup provide karte hain - dictionary system! 📖
  • Collections utility class useful methods provide karta hai collections pe operate karne ke liye - helper functions! 🔧
  • Most collection implementations by default thread-safe nahi hoti; multi-threaded environments ke liye synchronized wrappers ya concurrent collections use karo! 🧵
  • Hamesha collections ke saath generics use karo type safety ensure karne ke liye - compile-time type checking! 🔒
