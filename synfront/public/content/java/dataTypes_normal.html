Data Types and Variables

In Java, data types specify what kind of data a variable can hold, and variables are used to store and manipulate data.

Variables in Java

A variable is a container that holds a value. To use a variable in Java, you must:

  1. Declare the variable (specify its type and name)
  2. Initialize the variable (assign a value)
// Declaration
int age;

// Initialization
age = 25;

// Declaration and initialization in one statement
int height = 180;

Primitive Data Types

Java has eight primitive data types that store simple values:

Data Type Size Range Example
byte 8 bits -128 to 127 byte b = 100;
short 16 bits -32,768 to 32,767 short s = 1000;
int 32 bits -2^31 to 2^31-1 int i = 100000;
long 64 bits -2^63 to 2^63-1 long l = 100000L;
float 32 bits ~±3.40282347E+38F float f = 3.14f;
double 64 bits ~±1.79769313486231570E+308 double d = 3.14159;
char 16 bits 0 to 65,535 char c = 'A';
boolean 1 bit true or false boolean b = true;

Reference Data Types

Reference types store references to objects. Some common reference types include:

  • Classes (e.g., String, Scanner)
  • Arrays
  • Interfaces
  • Enum
// String (reference type)
String name = "John";

// Array (reference type)
int[] numbers = {1, 2, 3, 4, 5};

Type Conversion

Java supports two types of conversion between data types:

Implicit Conversion (Widening)

Automatic conversion from smaller to larger data types:

byte b = 100;
int i = b;  // Automatic conversion from byte to int

Explicit Conversion (Narrowing)

Manual conversion from larger to smaller data types using casting:

double d = 100.25;
int i = (int)d;  // Explicit casting, i will be 100 (decimal part is truncated)

Constants

Constants are variables whose values cannot be changed after initialization. In Java, constants are declared using the final keyword:

final double PI = 3.14159;
final int MAX_USERS = 100;

Variable Scope

The scope of a variable defines where in the code the variable is accessible:

Class Variables (Static Fields)

public class MyClass {
    static int classVar = 10;  // Class variable
}

Instance Variables (Non-static Fields)

public class MyClass {
    int instanceVar = 20;  // Instance variable
}

Local Variables

public void myMethod() {
    int localVar = 30;  // Local variable
}

Block Variables

public void myMethod() {
    if (true) {
        int blockVar = 40;  // Block variable
    }
    // blockVar is not accessible here
}

Key points to remember:

  • Java is a statically-typed language, meaning variable types must be declared
  • Primitive types store actual values, reference types store addresses
  • Use the appropriate data type for your needs to optimize memory usage
  • Be aware of potential data loss during type conversion
  • Use constants (final) for values that should not change
  • Understand variable scope to avoid unexpected behavior
  • Java initializes class and instance variables with default values, but local variables must be explicitly initialized before use
