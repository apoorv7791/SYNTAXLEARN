Setting Up Your Environment

To start writing Java code, you need to set up your development environment. Here are the tools you need:

  • Java Development Kit (JDK): The essential software package for Java development that includes the Java Runtime Environment (JRE), compiler, and other tools
  • Integrated Development Environment (IDE): Software that provides comprehensive facilities for Java development

Installing the JDK

Follow these steps to install the JDK:

  1. Visit the Oracle JDK download page or use AdoptOpenJDK for an open-source alternative
  2. Download the latest version of the JDK for your operating system
  3. Run the installer and follow the installation instructions
  4. Set up the JAVA_HOME environment variable pointing to your JDK installation directory
  5. Add the JDK's bin directory to your system's PATH variable

Verifying the Installation

To verify that Java is installed correctly, open a terminal or command prompt and type:

java -version
javac -version

You should see output showing the installed Java version for both commands.

Choosing an IDE

Several IDEs are available for Java development:

  • IntelliJ IDEA: A powerful and user-friendly IDE with a free Community Edition
  • Eclipse: A widely used open-source IDE with extensive plugin support
  • NetBeans: A free, open-source IDE that simplifies Java development
  • Visual Studio Code: A lightweight code editor with Java extensions

Your First Java Program

Once your environment is set up, you can write your first Java program:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Save this code in a file named HelloWorld.java. To compile and run the program:

// Compile the program
javac HelloWorld.java

// Run the program
java HelloWorld

You should see the output: Hello, World!

Understanding the Hello World Program

  • public class HelloWorld: Defines a class named HelloWorld. In Java, every application must contain at least one class, and the class name must match the filename.
  • public static void main(String[] args): The main method is the entry point of a Java application. The Java runtime starts execution from this method.
  • System.out.println("Hello, World!");: Outputs the text "Hello, World!" to the console.

Key points to remember:

  • Java is case-sensitive; HelloWorld and helloworld are different identifiers
  • Every Java application must have a main method as its entry point
  • Java source files must have the .java extension
  • The compiled bytecode files have the .class extension
  • The class name must match the filename (e.g., HelloWorld.java contains a class named HelloWorld)
