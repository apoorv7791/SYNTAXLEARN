Control Flow - Code ko Control Karna! 🚦

Bhai, Control Flow statements se hum code ke flow ko control karte hain - kya execute hoga, kya nahi, kitni baar hoga - sab kuch! Bilkul traffic signals ki tarah! 🚥

Decision Making Statements - Faisla Lene Wale! 🤔

if Statement - Agar Wala Scene! ✅

if statement ek condition check karta hai, aur agar woh true hai toh code execute hota hai:

int age = 20;

if (age >= 18) {
    System.out.println("Bhai, tu adult hai!");  // Condition true hai toh yeh print hoga
}

if-else Statement - Agar-Warna Wala Scene! 🔄

if-else statement mein, agar condition true hai toh ek code, warna dusra code execute hota hai:

int age = 15;

if (age >= 18) {
    System.out.println("Bhai, tu adult hai!");
} else {
    System.out.println("Abey bacche, abhi time hai!");  // Yeh print hoga
}

if-else-if Ladder - Multiple Conditions Check Karna! 🪜

if-else-if ladder se hum multiple conditions ek ke baad ek check kar sakte hain:

int score = 85;

if (score >= 90) {
    System.out.println("Grade: A - Topper hai tu!");
} else if (score >= 80) {
    System.out.println("Grade: B - Not bad bhai!");  // Yeh print hoga
} else if (score >= 70) {
    System.out.println("Grade: C - Thik hai yaar!");
} else if (score >= 60) {
    System.out.println("Grade: D - Bach gaya tu!");
} else {
    System.out.println("Grade: F - Agle saal milte hain!");
}

switch Statement - Menu Card Wala Scene! 📋

switch statement ek variable ki value check karke uske hisaab se code execute karta hai - bilkul menu card se order karne jaisa:

int day = 4;
String dayName;

switch (day) {
    case 1:
        dayName = "Monday - Somwar ko dukhi mat ho!";
        break;
    case 2:
        dayName = "Tuesday - Mangalwar shubh hai!";
        break;
    case 3:
        dayName = "Wednesday - Budhwar, mid-week chill!";
        break;
    case 4:
        dayName = "Thursday - Weekend ka countdown shuru!";  // Yeh select hoga
        break;
    case 5:
        dayName = "Friday - Weekend aa gaya!";
        break;
    case 6:
        dayName = "Saturday - Party time!";
        break;
    case 7:
        dayName = "Sunday - Aaram ka din!";
        break;
    default:
        dayName = "Bhai, calendar dekh le!";
}
System.out.println(dayName);

Java 12 ke baad, fancy switch expression bhi use kar sakte ho:

int day = 4;
String dayName = switch (day) {
    case 1 -> "Monday - Somwar blues!";
    case 2 -> "Tuesday - Mangalwar!";
    case 3 -> "Wednesday - Mid-week!";
    case 4 -> "Thursday - Weekend coming!";  // Yeh select hoga
    case 5 -> "Friday - Weekend start!";
    case 6 -> "Saturday - Party mode on!";
    case 7 -> "Sunday - Chill mode on!";
    default -> "Bhai, calendar dekh le!";
};
System.out.println(dayName);

Looping Statements - Ghoomne Wale! 🔄

for Loop - Fixed Number of Rounds! 🔢

for loop ek code block ko fixed number of times execute karta hai:

// 1 se 5 tak print karo
for (int i = 1; i <= 5; i++) {
    System.out.println(i + " baar ho gaya!");
}

Enhanced for Loop (for-each) - Collection pe Loop! 📚

Enhanced for loop arrays aur collections pe loop karne ka shortcut hai:

int[] numbers = {1, 2, 3, 4, 5};

// Array ke har element ko print karo
for (int num : numbers) {
    System.out.println("Number: " + num);
}

while Loop - Jab Tak Condition True Hai! ⏱️

while loop tab tak code execute karta hai jab tak condition true hai:

int i = 1;

// 1 se 5 tak print karo
while (i <= 5) {
    System.out.println(i + " - while loop chal raha hai!");
    i++;  // Increment karna mat bhoolna, warna infinite loop ho jayega!
}

do-while Loop - Kam Se Kam Ek Baar! 🔄

do-while loop while loop ki tarah hai, par kam se kam ek baar code execute zaroor hota hai, phir condition check hoti hai:

int i = 1;

// 1 se 5 tak print karo
do {
    System.out.println(i + " - pehle execute, baad mein check!");
    i++;
} while (i <= 5);

Jump Statements - Koodne Wale! 🦘

break Statement - Loop Se Bahar Niklo! 🚪

break statement loop ya switch se turant bahar nikal deta hai:

// 1 se 10 tak print karo, par 5 pe rok do
for (int i = 1; i <= 10; i++) {
    if (i > 5) {
        System.out.println("Bas bhai, 5 ke baad exit kar rahe hain!");
        break;  // Loop se bahar nikal jao jab i > 5
    }
    System.out.println(i);
}

continue Statement - Current Iteration Skip Karo! ⏭️

continue statement current iteration ko skip karke next iteration pe chala jata hai:

// 1 se 10 tak odd numbers print karo
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        System.out.println(i + " even hai, skip kar rahe hain!");
        continue;  // Even numbers ko skip karo
    }
    System.out.println(i + " odd hai, print kar rahe hain!");
}

return Statement - Method Se Bahar Niklo! 🏃‍♂️

return statement method se bahar nikal deta hai, aur optional value return karta hai:

public boolean isEven(int number) {
    if (number % 2 == 0) {
        return true;  // Method se exit karo aur true return karo
    }
    return false;  // Method se exit karo aur false return karo
}

Yaad Rakhne Wali Baatein! 🧠

  • if statements use karo jab conditions ke basis pe decision lena ho! 🤔
  • switch statements use karo jab ek variable ki multiple values check karni ho! 📋
  • for loops use karo jab pehle se pata ho kitni baar loop chalana hai! 🔢
  • while loops use karo jab iterations ki count condition pe depend kare! ⏱️
  • do-while loops use karo jab code kam se kam ek baar execute karna ho! 🔄
  • break use karo loop ya switch se turant bahar nikalne ke liye! 🚪
  • continue use karo current iteration skip karne ke liye! ⏭️
  • Hamesha curly braces {} use karo code blocks ke liye, single statements ke liye bhi - bugs se bachega! 🐞
  • Infinite loops se bacho (loops jo kabhi terminate nahi hote) - program hang ho jayega! ♾️
