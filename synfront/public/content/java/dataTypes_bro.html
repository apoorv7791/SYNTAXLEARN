Data Types aur Variables - Java ke Dabbo! 📦

Bhai, Java mein data types aur variables samajhna ekdum zaroori hai! Yeh woh boxes hain jisme hum apna data store karte hain! 🎁

Variables Kya Hote Hain? 🤔

Variable ek aisa box hai jisme hum koi value store kar sakte hain. Java mein variable use karne ke liye:

  1. Pehle declare karo (type aur name batao) - box banao aur label lagao! 📋
  2. Phir initialize karo (value assign karo) - box mein saman rakho! 🧳
// Declaration - box banaya
int age;

// Initialization - box mein value daali
age = 25;

// Ek hi line mein dono kaam
int height = 180;  // Smart move! 😎

Primitive Data Types - Basic Boxes! 📦

Java mein 8 primitive data types hain - yeh simple values store karte hain:

Data Type Size Range Example
byte 8 bits -128 to 127 byte b = 100;
short 16 bits -32,768 to 32,767 short s = 1000;
int 32 bits -2^31 to 2^31-1 int i = 100000;
long 64 bits -2^63 to 2^63-1 long l = 100000L;
float 32 bits Decimal numbers float f = 3.14f;
double 64 bits Bade decimal numbers double d = 3.14159;
char 16 bits Single characters char c = 'A';
boolean 1 bit true ya false boolean b = true;

Reference Data Types - Fancy Boxes! 🎀

Reference types objects ke addresses store karte hain - yeh actual boxes ke pate hain! 🏠

  • Classes (jaise String, Scanner) - pre-made fancy boxes! 🏢
  • Arrays - boxes ka collection! 📚
  • Interfaces - contracts! 📜
  • Enum - fixed options ka list! 📋
// String (reference type) - text ka box
String name = "Bhai";

// Array (reference type) - boxes ka collection
int[] numbers = {1, 2, 3, 4, 5};  // 5 boxes ek saath!

Type Conversion - Box Badalna! 🔄

Java mein do tarah se data types convert hote hain:

Implicit Conversion (Widening) - Automatic Upgrade! ⬆️

Chote box se bade box mein automatic conversion:

byte b = 100;  // Chota box
int i = b;    // Bada box - automatic conversion ho gaya!

Explicit Conversion (Narrowing) - Manual Downgrade! ⬇️

Bade box se chote box mein manual conversion (casting):

double d = 100.25;     // Bada box
int i = (int)d;        // Chota box - manually cast karna pada
                      // i ki value 100 hogi, .25 gayab! 💨

Constants - Fixed Boxes! 🔒

Constants woh variables hain jinki value kabhi change nahi ho sakti. Java mein final keyword se constants banate hain:

final double PI = 3.14159;      // Yeh value kabhi change nahi hogi
final int MAX_USERS = 100;      // Isko lock kar diya gaya hai!

Variable Scope - Kahan Tak Chalta Hai? 🗺️

Variable ka scope batata hai ki woh kahan-kahan accessible hai:

Class Variables (Static Fields) - Sabke Liye Common! 🌍

public class MyClass {
    static int classVar = 10;  // Class variable - sabke liye same!
}

Instance Variables (Non-static Fields) - Har Object Ka Apna! 🏠

public class MyClass {
    int instanceVar = 20;  // Instance variable - har object ka personal!
}

Local Variables - Method ke Andar! 🏢

public void myMethod() {
    int localVar = 30;  // Local variable - sirf is method mein available!
}

Block Variables - Curly Braces ke Andar! 🧱

public void myMethod() {
    if (true) {
        int blockVar = 40;  // Block variable - sirf in curly braces ke andar!
    }
    // blockVar yahan available nahi hai - out of scope! 🚫
}

Yaad Rakhne Wali Baatein! 🧠

  • Java statically-typed language hai - har variable ka type pehle se declare karna padta hai! 📝
  • Primitive types actual values store karte hain, reference types addresses store karte hain! 🏠
  • Memory bachane ke liye sahi data type choose karo - int ke liye byte use mat karo! 💾
  • Type conversion mein data loss ho sakta hai - dhyan rakhna! ⚠️
  • Jo values change nahi honi chahiye, unhe final banao! 🔒
  • Variable scope samajho - galat jagah variable access karne ki koshish mat karo! 🚫
  • Class aur instance variables ko Java default values de deta hai, par local variables ko explicitly initialize karna padta hai! ⚙️
