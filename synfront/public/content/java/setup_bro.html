Ab setup ho gaya hai toh chalo pehla program likhte hain - Hello World! Classic hai bhai!

Hello World Program Samjho! ğŸ§ 

  • public class HelloWorld: Yeh ek class define karta hai "HelloWorld" naam ki. Java mein har program kam se kam ek class mein hona chahiye, aur filename aur class ka naam same hona chahiye!
  • public static void main(String[] args): Yeh main method hai - har Java program ka starting point! Program yahan se execute hona start hota hai!
  • System.out.println("Hello, World!");: Yeh line console pe "Hello, World!" print karti hai - output dikhane ke liye!
