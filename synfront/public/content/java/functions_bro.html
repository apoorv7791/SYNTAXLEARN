Method Overloading - Ek Naam, Multiple Versions! ğŸ”„

Method overloading se same naam ke multiple methods bana sakte hain, bas parameters alag hone chahiye:

public class Calculator {
    // Do integers ke liye add method
    public int add(int a, int b) {
        return a + b;
    }
    
    // Teen integers ke liye add method - overloaded!
    public int add(int a, int b, int c) {
        return a + b + c;
    }
    
    // Double numbers ke liye add method - ye bhi overloaded!
    public double add(double a, double b) {
        return a + b;
    }
    
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        
        System.out.println(calc.add(5, 3));         // Pehla method call hoga: 8
        System.out.println(calc.add(5, 3, 2));      // Dusra method call hoga: 10
        System.out.println(calc.add(5.5, 3.5));     // Teesra method call hoga: 9.0
    }
}

Primitive Types Pass Karna ğŸ”¢

Jab primitive types (int, float, etc.) pass karte hain, toh value ka copy pass hota hai (pass-by-value):

public void modifyValue(int x) {
    x = x + 10;  // Local copy modify hoga, original nahi
}

public static void main(String[] args) {
    int num = 5;
    modifyValue(num);
    System.out.println(num);  // Still prints 5, change nahi hua!
}

Reference Types Pass Karna ğŸ“¦

Jab objects pass karte hain, toh reference ka copy pass hota hai, jisse actual object modify ho sakta hai:

public void modifyArray(int[] arr) {
    arr[0] = 100;  // Actual array modify hoga
}

public static void main(String[] args) {
    int[] numbers = {1, 2, 3};
    modifyArray(numbers);
    System.out.println(numbers[0]);  // Prints 100, change ho gaya!
}

Recursion - Method Khud Ko Call Kare! ğŸ”„

Recursion mein method khud ko call karta hai - Inception movie jaisa scene! ğŸ¬

public int factorial(int n) {
    // Base case - rukne ki condition
    if (n == 0 || n == 1) {
        return 1;
    }
    // Recursive case - khud ko hi call kiya!
    return n * factorial(n - 1);
}

public static void main(String[] args) {
    Calculator calc = new Calculator();
    System.out.println(calc.factorial(5));  // 5! = 5 * 4 * 3 * 2 * 1 = 120
}
