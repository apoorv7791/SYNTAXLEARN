Break aur Continue - Special Controls! ğŸ›‘

Break aur continue se loops ko aur bhi control kar sakte hain:

  • break: Loop se ekdum bahar nikal jao! Jaise movie beech mein chhod ke chale jaana! ğŸšª
  • continue: Current iteration skip karke next iteration pe jump karo! Jaise class mein ek chapter skip karna! â­ï¸
// Break example
for (int i = 1; i <= 10; i++) {
    if (i == 6) {
        break;  // Jab i = 6 ho jaye, loop se bahar nikal jao
    }
    cout << i << " ";
}
// Output: 1 2 3 4 5

// Continue example
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // Even numbers ko skip karo
    }
    cout << i << " ";
}
// Output: 1 3 5 7 9
