Control Flow - Program Ko Control Karna! 🚦

Bhai, control flow statements se hum decide karte hain ki program mein kaunsa code kab chalega! Traffic signal ki tarah - kabhi rukna hai, kabhi chalna hai! 🚥

Conditional Statements - Conditions Check Karo! 🧐

Yeh statements decide karte hain ki code chalega ya nahi, based on conditions:

If Statement - Agar Condition True Hai Toh! ✅

if (condition) {
    // Yeh code tabhi chalega jab condition true hogi
}

If-Else Statement - Ya Toh, Ya Fir! 🔄

if (condition) {
    // Condition true hai toh yeh chalega
} else {
    // Condition false hai toh yeh chalega
}

If-Else If-Else Statement - Multiple Conditions! 🎯

if (condition1) {
    // Condition1 true hai toh yeh chalega
} else if (condition2) {
    // Condition1 false hai aur Condition2 true hai toh yeh chalega
} else {
    // Dono conditions false hain toh yeh chalega
}

Switch Statement - Multiple Cases Handle Karna! 🎛️

switch (expression) {
    case value1:
        // Value1 ke liye code
        break;
    case value2:
        // Value2 ke liye code
        break;
    default:
        // Koi bhi case match nahi hua toh yeh chalega
}

Dekho examples kaise kaam karte hain:

int score = 85;

if (score >= 90) {
    cout << "Grade: A";  // 90+ ke liye A grade
} else if (score >= 80) {
    cout << "Grade: B";  // 80-89 ke liye B grade
} else if (score >= 70) {
    cout << "Grade: C";  // 70-79 ke liye C grade
} else {
    cout << "Grade: F";  // 70 se kam ke liye F grade
}

// Switch ka example
int day = 3;
switch (day) {
    case 1:
        cout << "Monday";  // Day 1 = Monday
        break;
    case 2:
        cout << "Tuesday";  // Day 2 = Tuesday
        break;
    case 3:
        cout << "Wednesday";  // Day 3 = Wednesday
        break;
    default:
        cout << "Other day";  // Baaki din
}

Loops - Code Ko Baar Baar Chalana! 🔄

Loops se hum same code ko multiple times chala sakte hain, baar baar likhne ki zaroorat nahi!

For Loop - Jab Pata Ho Kitni Baar Chalana Hai! 🔢

for (initialization; condition; update) {
    // Yeh code baar baar chalega
}

While Loop - Jab Tak Condition True Hai Tab Tak Chalao! ⏱️

while (condition) {
    // Jab tak condition true hai, tab tak chalega
}

Do-While Loop - Kam Se Kam Ek Baar Toh Chalega Hi! 🔁

do {
    // Pehle ek baar chalega, fir condition check karega
} while (condition);

Loops ke examples dekho:

// For loop se 1 se 5 tak print karna
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// Output: 1 2 3 4 5

// While loop se 1 se 5 tak print karna
int j = 1;
while (j <= 5) {
    cout << j << " ";
    j++;  // j ko update karna mat bhoolna!
}
// Output: 1 2 3 4 5

// Do-while loop se 1 se 5 tak print karna
int k = 1;
do {
    cout << k << " ";
    k++;
} while (k <= 5);
// Output: 1 2 3 4 5

Break aur Continue - Special Controls! 🛑

Break aur continue se loops ko aur bhi control kar sakte hain:

  • break: Loop se ekdum bahar nikal jao! Jaise movie beech mein chhod ke chale jaana! 🚪
  • continue: Current iteration skip karke next iteration pe jump karo! Jaise class mein ek chapter skip karna! ⏭️
// Break example
for (int i = 1; i <= 10; i++) {
    if (i == 6) {
        break;  // Jab i = 6 ho jaye, loop se bahar nikal jao
    }
    cout << i << " ";
}
// Output: 1 2 3 4 5

// Continue example
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // Even numbers ko skip karo
    }
    cout << i << " ";
}
// Output: 1 3 5 7 9

Yaad Rakhne Wali Baatein! 🧠

  • Code blocks ke liye hamesha braces {} use karo, chahe ek hi line ho! Safety first! 🛡️
  • Loop conditions dhyan se likho, warna infinite loop ho jayega aur program hang ho jayega! 🔄
  • Apne use case ke hisaab se sahi loop choose karo! 🎯
  • Switch cases mein break statement likhna mat bhoolna, warna code neeche flow ho jayega! ⚠️
  • Nested loops aur conditionals use kar sakte ho, par complexity badh jayegi! 🧩
