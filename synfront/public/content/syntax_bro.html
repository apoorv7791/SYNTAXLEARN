C++ ka Basic Structure aur Syntax! 🚀

Bhai, C++ ek khass tarike se likha jata hai! Agar yeh basics samajh gaye, toh aadha game jeet gaye! 😎

Program ka Structure

Ek basic C++ program kuch aise dikhta hai:

#include <iostream>
using namespace std;

int main() {
    // Tera code yahan likhega
    cout << "Hello, World!" << endl;
    return 0;
}

Ab isko thoda samajhte hain:

  • #include <iostream>: Bhai yeh input/output wala toolbox hai, iske bina print nahi kar payega! 🧰
  • using namespace std;: Isse baar-baar std:: likhne se bachega, shortcut samajh le! 🏃‍♂️
  • int main(): Yeh main function hai, program yahan se shuru hota hai, bilkul film ke hero ki tarah! 🦸‍♂️
  • return 0;: Program successful raha toh 0 return karta hai, matlab "mission accomplished"! ✅

// Yeh single-line comment hai, sirf ek line

/* Yeh multi-line comment hai
   Jitna marzi likho,

Statements aur Semicolons

C++ mein har statement ke baad semicolon lagana zaroori hai, bilkul sentence ke end mein full-stop ki tarah:

int x = 5;  // Variable declare kiya
cout << x;  // Output kiya
x = x + 1;  // Value update kiya

Code ke Blocks

Curly braces {} se hum multiple statements ko ek group mein rakhte hain:

if (x > 0) {
    cout << "Positive hai bhai!";
    x = 0;  // Reset kar diya
}

Yaad rakhne wali baatein:

  • Har C++ program mein main() function hona chahiye, warna program start hi nahi hoga! 🚫
  • Har statement ke baad semicolon lagana mat bhoolna, warna error aayega! ⚠️
  • Code blocks curly braces {} mein likhte hain, jaise kisi box mein pack karna! 📦
  • C++ case-sensitive hai, matlab 'Variable' aur 'variable' alag-alag cheezein hain! 🔤
  • Whitespace (spaces, tabs, newlines) ko compiler ignore karta hai, par code readable rakhne ke liye use karo! 👀
