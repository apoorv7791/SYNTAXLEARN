Statements aur Semicolons

C++ mein har statement ke baad semicolon lagana zaroori hai, bilkul sentence ke end mein full-stop ki tarah:

int x = 5;  // Variable declare kiya
cout << x;  // Output kiya
x = x + 1;  // Value update kiya

Code ke Blocks

Curly braces {} se hum multiple statements ko ek group mein rakhte hain:

if (x > 0) {
    cout << "Positive hai bhai!";
    x = 0;  // Reset kar diya
}
