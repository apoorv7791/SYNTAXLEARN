Function Declaration - Function Ko Define Karna

JavaScript mein functions define karne ke kai tarike hain:

Function Declaration - Purana Tarika

// Syntax
function functionName(parameter1, parameter2, ...) {
    // function body
    return value;  // optional
}

// Example
function greet(name) {
    return `Hello, ${name} bro!`;
}

console.log(greet("Rahul"));  // Output: "Hello, Rahul bro!"

Function Expression - Variable Mein Store Karna

// Syntax
const functionName = function(parameter1, parameter2, ...) {
    // function body
    return value;  // optional
};

// Example
const greet = function(name) {
    return `Hello, ${name} bro!`;
};

console.log(greet("Rahul"));  // Output: "Hello, Rahul bro!"

Arrow Functions - Modern Cool Tarika (ES6)

// Syntax
const functionName = (parameter1, parameter2, ...) => {
    // function body
    return value;  // optional
};

// Example
const greet = (name) => {
    return `Hello, ${name} bro!`;
};

// Shorter syntax for single expressions - One-liner!
const greet = name => `Hello, ${name} bro!`;

console.log(greet("Rahul"));  // Output: "Hello, Rahul bro!"
Note bro: Arrow functions normal functions se thode alag hain:
  • Inke paas apna this nahi hota (parent ka this use karte hain)
  • Inko new keyword ke saath use nahi kar sakte
  • Inke paas arguments object nahi hota
  • Inke paas apna super ya new.target nahi hota

Function Parameters - Function Ko Data Pass Karna

Functions ko data pass karne ke liye parameters use karte hain - jaise ki biryani mein masale!

Default Parameters - Default Values

// Syntax
function functionName(parameter1 = defaultValue1, parameter2 = defaultValue2, ...) {
    // function body
}

// Example
function greet(name = "Stranger") {
    return `Hello, ${name} bro!`;
}

console.log(greet());        // Output: "Hello, Stranger bro!"
console.log(greet("Rahul"));  // Output: "Hello, Rahul bro!"

Rest Parameters - Multiple Arguments

// Syntax
function functionName(...parameterName) {
    // function body
}

// Example
function addAllNumbers(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

console.log(addAllNumbers(1, 2, 3, 4, 5));  // Output: 15
console.log(addAllNumbers(10, 20, 30));     // Output: 60

Function Return - Function Se Value Wapas Lena

Functions return statement ke through value wapas kar sakte hain. Agar koi return statement nahi hai, toh function undefined return karta hai.

// Example with return
function multiply(a, b) {
    return a * b;  // Value ko wapas karta hai
}

console.log(multiply(5, 3));  // Output: 15

// Example without return
function logMessage(message) {
    console.log(message);
    // No return statement, so it returns undefined
}

const result = logMessage("Kya scene hai bro?");
console.log(result);  // Output: undefined

Function Scope - Function Ka Private Area

Function ke andar define kiye gaye variables bahar se access nahi kiye ja sakte. Ye function scope kehlaata hai - jaise ki ghar ke andar ki baatein ghar ke andar hi rehti hain!

// Example
function myFunction() {
    const localVar = "Main sirf function ke andar available hoon";
    console.log(localVar);  // Accessible
}

myFunction();
// console.log(localVar);  // Error: localVar is not defined - bahar se access nahi kar sakte!

Closures - Function Ke Andar Function

Closure ek aisa function hai jo apne outer function ke variables ko access kar sakta hai, even after outer function return ho chuka ho. Ye thoda magic jaisa hai! ü™Ñ

// Example
function createCounter() {
    let count = 0;  // Private variable
    
    return function() {
        count++;  // Outer function ke variable ko access kar raha hai
        return count;
    };
}

const counter = createCounter();
console.log(counter());  // Output: 1
console.log(counter());  // Output: 2
console.log(counter());  // Output: 3

Immediately Invoked Function Expressions (IIFE) - Turant Execute Hone Wale Functions

IIFE ek aisa function hai jo define hote hi execute ho jata hai - jaise ki instant noodles! üçú

// Syntax
(function() {
    // code
})();

// Example
(function() {
    const message = "IIFE se hello bro!";
    console.log(message);
})();  // Output: "IIFE se hello bro!"

Higher-Order Functions - Smart Functions

Higher-order functions wo functions hain jo doosre functions ko as arguments lete hain ya phir functions return karte hain. Ye functions ke boss hain! üòé

Functions as Arguments - Functions Ko Parameters Ke Tarah Use Karna

// Example
function executeOperation(x, y, operation) {
    return operation(x, y);
}

function add(a, b) {
    return a + b;
}

function multiply(a, b) {
    return a * b;
}

console.log(executeOperation(5, 3, add));      // Output: 8
console.log(executeOperation(5, 3, multiply));  // Output: 15

Functions Returning Functions - Functions Jo Functions Return Karte Hain

// Example
function createMultiplier(factor) {
    return function(number) {
        return number * factor;
    };
}

const double = createMultiplier(2);  // 2 se multiply karne wala function
const triple = createMultiplier(3);  // 3 se multiply karne wala function

console.log(double(5));  // Output: 10
console.log(triple(5));  // Output: 15

Common Built-in Higher-Order Functions - JavaScript Ke Ready-Made Smart Functions

JavaScript arrays ke saath kuch built-in higher-order functions aate hain jo life bahut easy bana dete hain:

Array.prototype.map() - Har Element Par Function Apply Karna

// Har element par function apply karke new array banata hai
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);
console.log(doubled);  // Output: [2, 4, 6, 8, 10]

Array.prototype.filter() - Elements Ko Filter Karna

// Condition pass karne wale elements ko new array mein rakhta hai
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers);  // Output: [2, 4]

Array.prototype.reduce() - Array Ko Single Value Mein Convert Karna

// Har element ko process karke ek single value return karta hai
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum);  // Output: 15

Array.prototype.forEach() - Har Element Par Function Run Karna

// Har element par function run karta hai, kuch return nahi karta
const numbers = [1, 2, 3, 4, 5];
numbers.forEach(num => console.log(`Number: ${num}`));
// Output:
// Number: 1
// Number: 2
// Number: 3
// Number: 4
// Number: 5

Recursion - Function Ka Khud Ko Call Karna

Recursion tab hota hai jab ek function khud ko call karta hai - jaise infinity mirror effect! ü™û

// Example: Factorial calculate karna
function factorial(n) {
    // Base case - rukne ki condition
    if (n <= 1) {
        return 1;
    }
    // Recursive case - khud ko call karna
    return n * factorial(n - 1);
}

console.log(factorial(5));  // Output: 120 (5 * 4 * 3 * 2 * 1)

Yaad Rakhne Wali Baatein

  • Functions code ke reusable pieces hain - ek baar likho, baar-baar use karo
  • Functions define karne ke 3 tarike hain: function declarations, function expressions, aur arrow functions
  • Arrow functions modern aur concise hain, lekin this ke mamle mein alag behave karte hain
  • Functions parameters le sakte hain aur values return kar sakte hain
  • Function ke andar define kiye gaye variables bahar se access nahi kiye ja sakte
  • Closures se functions apne outer scope ke variables ko access kar sakte hain
  • Higher-order functions doosre functions ko as arguments lete hain ya functions return karte hain
  • Array methods jaise map(), filter(), reduce() aur forEach() higher-order functions hain
  • Recursion tab hota hai jab function khud ko call karta hai
