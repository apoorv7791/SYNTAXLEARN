Conditional Statements - Faisla Lene Wale Statements

Ye statements aapko conditions ke basis par alag-alag code chalane dete hain.

if...else Statement - Agar...Warna Statement

Agar condition sahi hai, toh ek code chalao, warna doosra code chalao. Simple!

// Basic if statement
if (condition) {
    // tab ye code chalega jab condition true ho
}

// if...else statement
if (condition) {
    // tab ye code chalega jab condition true ho
} else {
    // tab ye code chalega jab condition false ho
}

// if...else if...else statement
if (condition1) {
    // tab ye code chalega jab condition1 true ho
} else if (condition2) {
    // tab ye code chalega jab condition1 false ho aur condition2 true ho
} else {
    // tab ye code chalega jab dono conditions false hon
}

Example:

const hour = new Date().getHours();
let greeting;

if (hour < 12) {
    greeting = "Good morning bro!";
} else if (hour < 18) {
    greeting = "Good afternoon bro!";
} else {
    greeting = "Good evening bro!";
}

console.log(greeting);

Ternary Operator - Shortcut Tarika

Ye ek shortcut hai if...else ka. Ek line mein kaam khatam!

// Syntax
condition ? expressionIfTrue : expressionIfFalse;

// Example
const age = 20;
const canVote = age >= 18 ? "Haan bro, vote de sakta hai" : "Nahi bro, abhi nahi";
console.log(canVote);  // Output: "Haan bro, vote de sakta hai"

switch Statement - Menu Jaisa Statement

Jab aapko ek variable ki multiple values check karni hon, tab switch use karo. Ye restaurant menu jaisa hai - aapne jo order kiya, wahi milega!

// Syntax
switch (expression) {
    case value1:
        // tab ye code chalega jab expression === value1
        break;
    case value2:
        // tab ye code chalega jab expression === value2
        break;
    // more cases...
    default:
        // tab ye code chalega jab koi match nahi hua
}

// Example
const day = new Date().getDay();
let dayName;

switch (day) {
    case 0:
        dayName = "Sunday - Chill day bro!";
        break;
    case 1:
        dayName = "Monday - Back to hustle!";
        break;
    case 2:
        dayName = "Tuesday - Still grinding!";
        break;
    case 3:
        dayName = "Wednesday - Halfway there!";
        break;
    case 4:
        dayName = "Thursday - Weekend ki planning!";
        break;
    case 5:
        dayName = "Friday - Party mode on!";
        break;
    case 6:
        dayName = "Saturday - Full masti!";
        break;
    default:
        dayName = "Pata nahi bro, koi alien day hai";
}

console.log(`Today is ${dayName}`);
Yaad rakhna bro: break statement bahut important hai switch mein. Agar break nahi lagaya, toh code agle case mein bhi ghus jayega, chahe condition match na ho! Jaise ki party mein uninvited guest aa jaye! ðŸ˜‚

Loops - Ghoomne Wale Statements

Loops ka matlab hai ek hi code ko baar-baar chalana, bina copy-paste kiye. Time bachao, code clean rakho!

for Loop - Counting Wala Loop

Jab aapko pata ho ki kitni baar loop chalana hai, tab for loop use karo.

// Syntax
for (initialization; condition; increment/decrement) {
    // ye code baar-baar chalega
}

// Example: 1 se 5 tak numbers print karo
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1, 2, 3, 4, 5

for...of Loop - Array Ke Liye Loop

Arrays aur strings par loop lagane ka sabse easy tarika!

// Syntax
for (variable of iterable) {
    // ye code baar-baar chalega
}

// Example: Array ke har item ko print karo
const fruits = ["apple", "banana", "cherry"];
for (const fruit of fruits) {
    console.log(`Mujhe ${fruit} pasand hai`);
}
// Output: "Mujhe apple pasand hai", "Mujhe banana pasand hai", "Mujhe cherry pasand hai"

for...in Loop - Object Ke Liye Loop

Objects ke properties par loop lagane ka tarika!

// Syntax
for (variable in object) {
    // ye code baar-baar chalega
}

// Example: Object ki properties print karo
const person = {
    name: "Rahul",
    age: 25,
    job: "Developer"
};

for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}
// Output: "name: Rahul", "age: 25", "job: Developer"
Dhyan do bro: for...in loop arrays ke liye use mat karo agar order important hai, kyunki ye properties ke order ko guarantee nahi karta. Array ke liye for...of ya normal for loop use karo.

while Loop - Tab Tak Chalao Jab Tak

Jab aapko pata na ho ki loop kitni baar chalega, tab while loop use karo.

// Syntax
while (condition) {
    // ye code tab tak chalega jab tak condition true hai
}

// Example: 1 se 5 tak numbers print karo
let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
// Output: 1, 2, 3, 4, 5

do...while Loop - Kam Se Kam Ek Baar Chalao

do...while loop kam se kam ek baar zaroor chalega, chahe condition false ho!

// Syntax
do {
    // ye code kam se kam ek baar chalega
} while (condition);

// Example: 1 se 5 tak numbers print karo
let j = 1;
do {
    console.log(j);
    j++;
} while (j <= 5);
// Output: 1, 2, 3, 4, 5

Loop Control Statements - Loop Ko Control Karne Wale Statements

Kabhi kabhi aapko loop ko beech mein hi rokna ya skip karna pad sakta hai.

break Statement - Emergency Exit

break statement loop se bahar nikal deta hai, jaise movie beech mein chhod ke chale jao!

// Example: Jab i 3 ho, tab loop se bahar niklo
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        console.log("3 aa gaya, mai nikal leta hoon!");
        break;
    }
    console.log(i);
}
// Output: 1, 2, "3 aa gaya, mai nikal leta hoon!"

continue Statement - Skip Karo

continue statement current iteration ko skip kar deta hai, jaise class mein teacher ko ignore karna!

// Example: Jab i 3 ho, tab skip karo
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        console.log("3 ko skip kar dete hain!");
        continue;
    }
    console.log(i);
}
// Output: 1, 2, "3 ko skip kar dete hain!", 4, 5

Error Handling - Errors Ko Handle Karna

Errors aane par program crash hone se bachane ke liye error handling use karo.

try...catch...finally Statement - Koshish Karo, Pakdo, Aur Finally

Jaise cricket mein catch drop hone par bhi game continue rehta hai, waise hi try...catch se program crash hone se bach jata hai.

// Syntax
try {
    // risky code jo error throw kar sakta hai
} catch (error) {
    // error ko handle karne wala code
} finally {
    // ye code hamesha chalega, chahe error aaye ya na aaye
}

// Example
try {
    // Koshish karo ek non-existent variable ko access karne ki
    console.log(nonExistentVariable.property);
} catch (error) {
    console.error("Oops! Error aa gaya bro:", error.message);
} finally {
    console.log("Ye line toh hamesha chalegi, chahe kuch bhi ho jaye!");
}

throw Statement - Khud Se Error Throw Karna

Kabhi kabhi aapko khud se error generate karna pad sakta hai.

// Example
function divide(a, b) {
    if (b === 0) {
        throw new Error("Bro, zero se divide nahi kar sakte! Physics ki dhajjiyaan mat udao!");
    }
    return a / b;
}

try {
    console.log(divide(10, 0));
} catch (error) {
    console.error(error.message);  // Output: "Bro, zero se divide nahi kar sakte! Physics ki dhajjiyaan mat udao!"
}

Yaad Rakhne Wali Baatein

  • Simple conditions ke liye if...else use karo
  • Multiple conditions check karne ke liye switch use karo
  • Ek line mein condition check karne ke liye ternary operator (?:) use karo
  • Sahi loop chunno:
    • for loop: jab pata ho kitni baar loop chalana hai
    • while loop: jab pata na ho kitni baar loop chalana hai
    • for...of loop: arrays ke liye
    • for...in loop: objects ke liye
  • Loop se jaldi bahar nikalne ke liye break use karo
  • Current iteration skip karne ke liye continue use karo
  • Errors handle karne ke liye hamesha try...catch blocks use karo
