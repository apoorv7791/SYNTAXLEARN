JavaScript ka Basic Syntax aur Structure! 🚀

Bhai, JavaScript ek flexible language hai jo C aur Java se inspire hui hai. Basic syntax aur structure samajhna zaroori hai taki sahi aur efficient code likh sake! 💪

JavaScript Statements - Code ke Sentences! 📝

JavaScript code statements se bana hota hai jo browser ya Node.js runtime execute karta hai. Statements ko semicolons (;) se separate karte hain:

// Yeh ek single statement hai
console.log("Hello, Bhai!");

// Multiple statements ek saath
let x = 5; let y = 10; let z = x + y;

Technically semicolons optional hain kyunki JavaScript automatic semicolon insertion (ASI) use karta hai, lekin good practice yahi hai ki semicolons lagao - errors se bachne ke liye! 🛡️

Comments - Code mein Notes! 📒

Comments se code mein notes add kar sakte ho jo JavaScript engine ignore karta hai:

// Yeh single-line comment hai - ek line ka note

/* Yeh multi-line comment hai
   Jitna marzi likho,
   Jitni lines chahe! */

Case Sensitivity - Uppercase/Lowercase Matter Karta Hai! 🔤

JavaScript case-sensitive hai. Variables, function names, aur baaki identifiers ko same capitalization se type karna padta hai:

let name = "Rahul";
let Name = "Rohit";  // Yeh name se alag variable hai

console.log(name);  // Output: Rahul
console.log(Name);  // Output: Rohit

Whitespace aur Line Breaks - Spaces Matter Nahi Karte! ⌨️

JavaScript whitespace (spaces, tabs, newlines) ko statements ke beech ignore karta hai. Isse code ko readable banane ke liye format kar sakte ho:

// Yeh dono same hain
let sum = x + y;

let sum=x+y;

Code Blocks - Code ke Paragraphs! 📋

Code blocks curly braces {} se define hote hain aur statements ko group karne ke liye use hote hain:

// If statement with a code block
if (paise_hain) {
    iPhone_kharido();
    party_karo();
    flex_karo();
}

Identifiers - Variables ke Naam! 🏷️

Identifiers variables, functions, classes, etc. ke liye use hote hain. Inke rules hain:

  • Letters, digits, underscores (_), aur dollar signs ($) ho sakte hain
  • Letter, underscore (_), ya dollar sign ($) se start hona chahiye
  • Reserved words (JavaScript keywords) use nahi kar sakte
  • Case-sensitive hote hain - name aur Name alag hain!
// Valid identifiers - sahi naam
let firstName = "Rahul";
let _privateVar = 42;
let $price = 19.99;
let counter1 = 0;

// Invalid identifiers - galat naam
// let 1stPlace = "Gold";  // Digit se start nahi kar sakte
// let for = "loop";       // Reserved word use nahi kar sakte

Variables - Data Store Karne ke Boxes! 📦

Variables data values store karne ke liye use hote hain. Modern JavaScript mein teen tarike hain variables declare karne ke:

// var use karke (purana tarika, function-scoped)
var age = 30;

// let use karke (block-scoped, reassign kar sakte hain)
let score = 75;

// const use karke (block-scoped, reassign nahi kar sakte)
const PI = 3.14159;

Pro tip: Default const use karo, aur sirf tab let use karo jab variable ko reassign karna ho. Modern JavaScript mein var se door raho! 🚫

Data Types - Data ke Prakar! 📊

JavaScript mein built-in data types hain:

Primitive Types - Basic Types! 🧱

// Number - integers aur decimals
let age = 30;
let price = 19.99;

// String - text data
let name = "Rahul";
let greeting = 'Namaste';

// Boolean - true/false values
let isActive = true;
let isCompleted = false;

// Undefined - value assign nahi hui
let undefinedVar;

// Null - intentionally khaali value
let emptyValue = null;

// Symbol (ES6) - unique identifiers
let uniqueId = Symbol("id");

// BigInt (ES2020) - bahut bade numbers
let bigNumber = 9007199254740991n;

Reference Types - Complex Types! 🏗️

// Object - properties ka collection
let person = {
    firstName: "Rahul",
    lastName: "Sharma",
    age: 30
};

// Array - values ka ordered collection
let colors = ["red", "green", "blue"];

// Function - reusable code block
function namaste() {
    console.log("Namaste Duniya!");
}

Operators - Calculations Karne ke Signs! ➗

JavaScript various operators support karta hai values pe operations perform karne ke liye:

Arithmetic Operators - Math Wale! ➕

let a = 10;
let b = 5;

let sum = a + b;        // Addition: 15
let difference = a - b; // Subtraction: 5
let product = a * b;    // Multiplication: 50
let quotient = a / b;   // Division: 2
let remainder = a % b;  // Modulus (remainder): 0
let power = a ** b;     // Exponentiation: 100000

Assignment Operators - Value Assign Karne Wale! 📝

let x = 10;  // 10 ko x mein assign karo

x += 5;      // Same as: x = x + 5
x -= 3;      // Same as: x = x - 3
x *= 2;      // Same as: x = x * 2
x /= 4;      // Same as: x = x / 4
x %= 3;      // Same as: x = x % 3

Comparison Operators - Compare Karne Wale! 🔍

let a = 5;
let b = 10;
let c = "5";

a == c;   // Equal to (sirf value): true
a === c;  // Strict equal to (value aur type dono): false
a != b;   // Not equal to: true
a !== c;  // Strict not equal to: true
a < b;    // Less than: true
a > b;    // Greater than: false
a <= b;   // Less than or equal to: true
a >= b;   // Greater than or equal to: false

Logical Operators - Conditions Combine Karne Wale! 🔗

let isAdult = true;
let hasPermission = false;

isAdult && hasPermission;  // Logical AND (dono true hone chahiye): false
isAdult || hasPermission;  // Logical OR (koi ek true hona chahiye): true
!isAdult;                  // Logical NOT (ulta kar do): false

Expressions - Value Calculate Karne Wale! 🧮

Expression code ka koi bhi valid part hai jo ek value mein resolve hota hai:

// Arithmetic expression
let result = 5 * (10 - 3);

// String expression
let fullName = firstName + " " + lastName;

// Logical expression
let canAccess = isAdult && hasPermission;

// Assignment expression
let x = y = z = 5;  // 5 ko z mein, fir y mein, fir x mein assign karta hai

Conditional Statements - Conditions Check Karne Wale! 🚦

Conditional statements alag-alag conditions ke basis pe alag-alag actions perform karne ke liye use hote hain:

if...else Statement - Agar-Warna Wala! 🤔

let age = 18;

if (age >= 18) {
    console.log("Bhai, tu adult hai!");
} else {
    console.log("Abhi baccha hai tu!");
}

switch Statement - Menu Card Jaisa! 📋

let day = 2;
let dayName;

switch (day) {
    case 1:
        dayName = "Monday - Somwar";
        break;
    case 2:
        dayName = "Tuesday - Mangalwar";
        break;
    case 3:
        dayName = "Wednesday - Budhwar";
        break;
    default:
        dayName = "Pata nahi kaunsa din hai!";
}

console.log(dayName);  // Output: Tuesday - Mangalwar

Loops - Code ko Repeat Karne Wale! 🔄

Loops code block ko multiple times execute karne ke liye use hote hain:

for Loop - Fixed Number of Rounds! 🔢

// 1 se 5 tak numbers print karo
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

while Loop - Condition True Hai Tab Tak! ⏱️

let i = 1;

// 1 se 5 tak numbers print karo
while (i <= 5) {
    console.log(i);
    i++;
}

do...while Loop - Kam Se Kam Ek Baar! 🔂

let i = 1;

// 1 se 5 tak numbers print karo
do {
    console.log(i);
    i++;
} while (i <= 5);

for...of Loop (arrays ke liye) 📚

const colors = ["red", "green", "blue"];

// Har color print karo
for (const color of colors) {
    console.log(color);
}

for...in Loop (objects ke liye) 🔑

const person = {
    name: "Rahul",
    age: 30,
    job: "Developer"
};

// Har property print karo
for (const key in person) {
    console.log(key + ": " + person[key]);
}

Functions - Reusable Code Blocks! 🧩

Functions code ke blocks hain jo particular task perform karte hain aur reuse kiye ja sakte hain:

// Function declaration - normal tarika
function namaste(name) {
    return "Namaste, " + name + "!";
}

// Function expression - variable mein store
const add = function(a, b) {
    return a + b;
};

// Arrow function (ES6) - shortcut tarika
const multiply = (a, b) => a * b;

// Functions ko call karna
console.log(namaste("Rahul"));   // Output: Namaste, Rahul!
console.log(add(5, 3));          // Output: 8
console.log(multiply(4, 2));     // Output: 8

Error Handling - Errors Se Deal Karna! 🚑

JavaScript try...catch blocks se errors handle karne ka tarika provide karta hai:

try {
    // Code jisme error aa sakta hai
    let result = koi_undefined_variable * 2;
} catch (error) {
    // Error handle karne ke liye code
    console.error("Oops! Error aa gaya:", error.message);
} finally {
    // Yeh code hamesha chalega, chahe error aaye ya na aaye
    console.log("Yeh to chalega hi chalega!");
}

Yaad Rakhne Wali Baatein! 🧠

  • JavaScript case-sensitive hai - name aur Name alag variables hain! 🔤
  • Statements ko semicolons (;) se separate karo - good practice hai! 📏
  • Comments // se single-line aur /* */ se multi-line likhte hain! 📝
  • Code blocks curly braces {} mein likhte hain! 📦
  • Default const use karo, reassign karna ho to let, aur var se door raho! 🚫
  • JavaScript mein primitive aur reference dono types hote hain! 📊
  • Hamesha strict equality (===) aur inequality (!==) operators use karo type coercion issues se bachne ke liye! 🎯
  • Functions JavaScript mein first-class objects hain - variables mein assign kar sakte hain, arguments ke tarah pass kar sakte hain! 🔝

Bas bhai, itna samajh gaya to JavaScript ka syntax master ho gaya tu! Ab code likhna shuru kar! 💻
