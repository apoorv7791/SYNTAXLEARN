SQL Syntax Basics

Yo bro! SQL (Structured Query Language) is how you talk to databases. It's like the universal language for data manipulation. Let's learn how to speak SQL like a boss!

Pro Tip: SQL commands aren't case-sensitive, but most devs write SQL keywords in ALL CAPS to make them stand out. It's like SHOUTING the important parts!

Database Operations

Creating a Database

Wanna create a new database? Just use the CREATE DATABASE command:

CREATE DATABASE database_name;

Example:

CREATE DATABASE school;

Selecting a Database

To tell the system which database you wanna work with:

-- MySQL, SQL Server
USE database_name;

-- PostgreSQL
\\c database_name

Example:

USE school;

Dropping a Database

Need to delete a database? The DROP DATABASE command has your back:

DROP DATABASE database_name;

Example:

DROP DATABASE old_school;

Warning Bro: DROP DATABASE is like deleting your entire Instagram account - everything's gone forever! Use it only when you're 100% sure.

Table Operations

Creating a Table

Tables are where all your data lives. Create one like this:

CREATE TABLE table_name (
    column1 datatype constraints,
    column2 datatype constraints,
    ...
);

Example:

CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    date_of_birth DATE,
    enrollment_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Altering a Table

Need to change your table? ALTER TABLE is your go-to command:

-- Add a column
ALTER TABLE table_name ADD column_name datatype constraints;

-- Modify a column
ALTER TABLE table_name MODIFY column_name new_datatype new_constraints;

-- Drop a column
ALTER TABLE table_name DROP COLUMN column_name;

Examples:

-- Add a phone number column
ALTER TABLE students ADD phone_number VARCHAR(20);

-- Change the size of the email column
ALTER TABLE students MODIFY email VARCHAR(150) UNIQUE;

-- Remove the date_of_birth column
ALTER TABLE students DROP COLUMN date_of_birth;

Dropping a Table

Wanna delete a table? Just drop it like it's hot:

DROP TABLE table_name;

Example:

DROP TABLE old_students;

Warning Bro: DROP TABLE permanently deletes the table and all its data. It's like accidentally deleting all your vacation photos - they're gone for good!

Data Manipulation

Inserting Data

Time to add some data to your tables:

-- Insert a single record
INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);

-- Insert multiple records at once
INSERT INTO table_name (column1, column2, ...) 
VALUES 
    (value1, value2, ...),
    (value1, value2, ...),
    ...;

Examples:

-- Insert a single student
INSERT INTO students (first_name, last_name, email) 
VALUES ('Rahul', 'Sharma', 'rahul.sharma@example.com');

-- Insert multiple students at once
INSERT INTO students (first_name, last_name, email) 
VALUES 
    ('Priya', 'Patel', 'priya.patel@example.com'),
    ('Amit', 'Kumar', 'amit.kumar@example.com'),
    ('Neha', 'Singh', 'neha.singh@example.com');

Selecting Data

Wanna see your data? SELECT is your best friend:

-- Get everything from a table
SELECT * FROM table_name;

-- Get only specific columns
SELECT column1, column2, ... FROM table_name;

-- Get data with conditions
SELECT column1, column2, ... FROM table_name WHERE condition;

-- Get data sorted in a specific order
SELECT column1, column2, ... FROM table_name ORDER BY column_name [ASC|DESC];

-- Get a limited number of records
SELECT column1, column2, ... FROM table_name LIMIT number;

Examples:

-- Get all students
SELECT * FROM students;

-- Get only names and emails
SELECT first_name, last_name, email FROM students;

-- Get students with last name 'Sharma'
SELECT * FROM students WHERE last_name = 'Sharma';

-- Get students sorted by last name (A to Z)
SELECT * FROM students ORDER BY last_name ASC;

-- Get only the first 5 students
SELECT * FROM students LIMIT 5;

Updating Data

Need to change some data? UPDATE has got you covered:

UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;

Example:

-- Update a student's email
UPDATE students SET email = 'new.email@example.com' WHERE id = 1;

-- Update multiple columns at once
UPDATE students SET first_name = 'Raj', last_name = 'Malhotra' WHERE id = 1;

Warning Bro: If you forget the WHERE clause in an UPDATE statement, you'll update EVERY SINGLE RECORD in the table! Always use a WHERE clause unless you really want to update everything.

Deleting Data

Wanna remove some records? DELETE is the way to go:

DELETE FROM table_name WHERE condition;

Example:

-- Delete a specific student
DELETE FROM students WHERE id = 1;

-- Delete all students with last name 'Sharma'
DELETE FROM students WHERE last_name = 'Sharma';

Warning Bro: If you forget the WHERE clause in a DELETE statement, you'll delete EVERY RECORD in the table! Always double-check your DELETE statements before running them.

Filtering Data

WHERE Clause

The WHERE clause is how you filter your data - it's like your search criteria:

SELECT column1, column2, ... FROM table_name WHERE condition;

Here are the operators you can use in your WHERE clause:

Operator What it means Example
= Equals WHERE age = 30
> Greater than WHERE age > 30
< Less than WHERE age < 30
>= Greater than or equal WHERE age >= 30
<= Less than or equal WHERE age <= 30
<> or != Not equal WHERE age <> 30
BETWEEN Between a range WHERE age BETWEEN 20 AND 30
LIKE Pattern matching WHERE name LIKE 'R%'
IN Matches any value in a list WHERE country IN ('India', 'USA', 'UK')
IS NULL Is empty/null WHERE phone IS NULL
IS NOT NULL Is not empty/null WHERE phone IS NOT NULL

Logical Operators

Need to combine multiple conditions? Use these logical operators:

Operator What it means Example
AND Both conditions must be true WHERE age > 20 AND country = 'India'
OR Either condition can be true WHERE age > 20 OR country = 'India'
NOT Reverses the condition WHERE NOT country = 'India'

Pattern Matching with LIKE

Need to search for patterns? LIKE is your best bet:

Wildcard What it does Example
% Matches any number of characters WHERE name LIKE 'R%' (names starting with R)
WHERE email LIKE '%@gmail.com' (Gmail addresses)
_ Matches exactly one character WHERE name LIKE 'R_j' (Raj, Roj, etc.)

Sorting and Limiting Results

ORDER BY Clause

Want your results in a specific order? ORDER BY is the way to go:

-- Sort in ascending order (A to Z, 1 to 10)
SELECT * FROM table_name ORDER BY column_name;

-- Sort in descending order (Z to A, 10 to 1)
SELECT * FROM table_name ORDER BY column_name DESC;

-- Sort by multiple columns
SELECT * FROM table_name ORDER BY column1 ASC, column2 DESC;

Examples:

-- Sort students by last name (A to Z)
SELECT * FROM students ORDER BY last_name;

-- Sort students by enrollment date (newest first)
SELECT * FROM students ORDER BY enrollment_date DESC;

-- Sort students by last name, then by first name
SELECT * FROM students ORDER BY last_name, first_name;

LIMIT Clause

Only want a certain number of results? LIMIT is your friend:

-- Get a specific number of records
SELECT * FROM table_name LIMIT number;

-- Get a specific number of records starting from a position
SELECT * FROM table_name LIMIT offset, count;  -- MySQL syntax
SELECT * FROM table_name LIMIT count OFFSET offset;  -- PostgreSQL syntax

Examples:

-- Get the first 5 students
SELECT * FROM students LIMIT 5;

-- Get 5 students, starting from the 6th student (MySQL)
SELECT * FROM students LIMIT 5, 5;

-- Get 5 students, starting from the 6th student (PostgreSQL)
SELECT * FROM students LIMIT 5 OFFSET 5;

Aggregate Functions

Need to calculate stuff from your data? These functions have your back:

Function What it does Example
COUNT() Counts the number of rows SELECT COUNT(*) FROM students;
SUM() Adds up all the values SELECT SUM(score) FROM exams;
AVG() Calculates the average SELECT AVG(score) FROM exams;
MIN() Finds the smallest value SELECT MIN(score) FROM exams;
MAX() Finds the largest value SELECT MAX(score) FROM exams;

GROUP BY Clause

Want to group your data and calculate stuff for each group? GROUP BY is what you need:

SELECT column1, aggregate_function(column2)
FROM table_name
GROUP BY column1;

Example:

-- Count how many students have each last name
SELECT last_name, COUNT(*) AS count
FROM students
GROUP BY last_name;

HAVING Clause

HAVING is like WHERE but for grouped data:

SELECT column1, aggregate_function(column2)
FROM table_name
GROUP BY column1
HAVING condition;

Example:

-- Find last names that appear more than once
SELECT last_name, COUNT(*) AS count
FROM students
GROUP BY last_name
HAVING count > 1;
