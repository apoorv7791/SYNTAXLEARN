Control Flow (if, for, while)

Control flow statements allow you to control the execution flow of your program based on conditions and loops.

Conditional Statements

Conditional statements execute code blocks based on whether a condition is true or false.

If Statement

if (condition) {
    // Code executes if condition is true
}

If-Else Statement

if (condition) {
    // Code executes if condition is true
} else {
    // Code executes if condition is false
}

If-Else If-Else Statement

if (condition1) {
    // Code executes if condition1 is true
} else if (condition2) {
    // Code executes if condition1 is false and condition2 is true
} else {
    // Code executes if both conditions are false
}

Switch Statement

switch (expression) {
    case value1:
        // Code for value1
        break;
    case value2:
        // Code for value2
        break;
    default:
        // Code if no case matches
}

Example of conditional statements:

int score = 85;

if (score >= 90) {
    cout << "Grade: A";
} else if (score >= 80) {
    cout << "Grade: B";
} else if (score >= 70) {
    cout << "Grade: C";
} else {
    cout << "Grade: F";
}

// Using switch
int day = 3;
switch (day) {
    case 1:
        cout << "Monday";
        break;
    case 2:
        cout << "Tuesday";
        break;
    case 3:
        cout << "Wednesday";
        break;
    default:
        cout << "Other day";
}

Loops

Loops allow you to execute a block of code repeatedly.

For Loop

for (initialization; condition; update) {
    // Code to repeat
}

While Loop

while (condition) {
    // Code to repeat as long as condition is true
}

Do-While Loop

do {
    // Code executes at least once
} while (condition);

Example of loops:

// For loop to print numbers 1 to 5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// Output: 1 2 3 4 5

// While loop to print numbers 1 to 5
int j = 1;
while (j <= 5) {
    cout << j << " ";
    j++;
}
// Output: 1 2 3 4 5

// Do-while loop to print numbers 1 to 5
int k = 1;
do {
    cout << k << " ";
    k++;
} while (k <= 5);
// Output: 1 2 3 4 5

Break and Continue

Break and continue statements provide additional control within loops:

  • break: Exits the loop immediately
  • continue: Skips the rest of the current iteration and moves to the next one
// Break example
for (int i = 1; i <= 10; i++) {
    if (i == 6) {
        break; // Exit loop when i is 6
    }
    cout << i << " ";
}
// Output: 1 2 3 4 5

// Continue example
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue; // Skip even numbers
    }
    cout << i << " ";
}
// Output: 1 3 5 7 9

Key points to remember:

  • Always use braces {} for code blocks, even for single statements
  • Be careful with loop conditions to avoid infinite loops
  • Use the appropriate loop type for your specific needs
  • Remember to include break statements in switch cases
  • Nested loops and conditionals are possible but use them carefully
