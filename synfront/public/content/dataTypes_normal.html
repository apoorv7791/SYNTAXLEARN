Data Types and Variables

In C++, data types specify what kind of data a variable can hold, and variables are used to store and manipulate data.

Basic Data Types

C++ has several built-in data types:

Data Type Description Size (bytes) Example
int Integer values 4 int age = 25;
float Single precision floating point 4 float price = 19.99f;
double Double precision floating point 8 double pi = 3.14159265359;
char Single character 1 char grade = 'A';
bool Boolean value (true/false) 1 bool isActive = true;

Variable Declaration and Initialization

Variables must be declared before they can be used:

// Declaration
int count;

// Declaration with initialization
int age = 25;
float salary = 50000.50;
char initial = 'J';
bool isEmployed = true;

Type Modifiers

Type modifiers can be used to modify the size and range of data types:

  • short: Reduces the size of an integer
  • long: Increases the size of an integer or double
  • signed: Allows positive and negative values (default)
  • unsigned: Allows only positive values
short int smallNumber = 32767;
long int bigNumber = 2147483647;
unsigned int positiveOnly = 4294967295;

Constants

Constants are variables whose values cannot be changed after initialization:

const double PI = 3.14159;
const int MAX_USERS = 100;

Type Conversion

C++ supports both implicit and explicit type conversion:

// Implicit conversion
int x = 10;
double y = x;  // int to double

// Explicit conversion (casting)
double a = 3.14;
int b = (int)a;  // double to int, b = 3
int c = static_cast<int>(a);  // Modern C++ casting

Key points to remember:

  • Always initialize variables before using them
  • Choose the appropriate data type for your needs
  • Be aware of potential data loss during type conversion
  • Use constants for values that should not change
  • Variable names are case-sensitive and should be descriptive
