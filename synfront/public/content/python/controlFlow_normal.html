Python Control Flow

Control flow refers to the order in which the statements in a program are executed. Python provides various control flow statements that allow you to control the flow of your program based on conditions, repeat blocks of code, and handle exceptions. This section covers the different control flow structures in Python.

Conditional Statements

Conditional statements allow you to execute different blocks of code based on whether a condition is true or false.

if Statement

The if statement is used to execute a block of code only if a specified condition is true.

Example: if Statement

age = 20

if age >= 18:
    print("You are an adult.")
    print("You can vote.")
Output:
You are an adult.
You can vote.

if-else Statement

The if-else statement allows you to execute one block of code if a condition is true and another block if the condition is false.

Example: if-else Statement

age = 15

if age >= 18:
    print("You are an adult.")
    print("You can vote.")
else:
    print("You are a minor.")
    print("You cannot vote yet.")
Output:
You are a minor.
You cannot vote yet.

if-elif-else Statement

The if-elif-else statement allows you to check multiple conditions and execute different blocks of code accordingly.

Example: if-elif-else Statement

score = 85

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
elif score >= 60:
    grade = 'D'
else:
    grade = 'F'

print(f"Your grade is {grade}")
Output:
Your grade is B
Note: Python uses indentation to define blocks of code. The standard indentation is 4 spaces, although any consistent indentation can be used. Be careful not to mix tabs and spaces.

Nested if Statements

You can place an if statement inside another if statement to create nested if statements.

Example: Nested if Statements

age = 25
has_license = True

if age >= 18:
    print("You are old enough to drive.")
    if has_license:
        print("You can drive legally.")
    else:
        print("You need to get a driver's license first.")
else:
    print("You are too young to drive.")
Output:
You are old enough to drive.
You can drive legally.

Conditional Expressions (Ternary Operator)

Python provides a concise way to write conditional statements in a single line, often called the ternary operator.

Example: Conditional Expressions

age = 20
status = "adult" if age >= 18 else "minor"
print(status)

# Equivalent to:
# if age >= 18:
#     status = "adult"
# else:
#     status = "minor"
Output:
adult

Loops

Loops allow you to execute a block of code repeatedly. Python provides two main types of loops: for loops and while loops.

for Loop

The for loop is used to iterate over a sequence (like a list, tuple, string, or range) and execute a block of code for each item in the sequence.

Example: for Loop with a List

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
Output:
apple
banana
cherry

Example: for Loop with range()

# range(5) generates numbers from 0 to 4
for i in range(5):
    print(i)

# range(start, stop) generates numbers from start to stop-1
for i in range(2, 6):
    print(i)

# range(start, stop, step) generates numbers from start to stop-1 with the given step
for i in range(1, 10, 2):
    print(i)
Output:
0
1
2
3
4
2
3
4
5
1
3
5
7
9

while Loop

The while loop executes a block of code as long as a specified condition is true.

Example: while Loop

count = 0

while count < 5:
    print(count)
    count += 1
Output:
0
1
2
3
4
Warning: Be careful with while loops! If the condition never becomes false, the loop will run indefinitely, creating an infinite loop. Always ensure that the condition will eventually become false.

Loop Control Statements

Python provides several statements to control the flow of loops:

break Statement

The break statement terminates the loop and transfers execution to the statement immediately following the loop.

Example: break Statement

for i in range(10):
    if i == 5:
        break
    print(i)
Output:
0
1
2
3
4

continue Statement

The continue statement skips the rest of the code inside the loop for the current iteration and moves to the next iteration.

Example: continue Statement

for i in range(10):
    if i % 2 == 0:  # Skip even numbers
        continue
    print(i)
Output:
1
3
5
7
9

else Clause in Loops

Both for and while loops can have an optional else clause that is executed when the loop is exhausted (i.e., when the loop condition becomes false), but not when the loop is terminated by a break statement.

Example: else Clause in for Loop

# Loop completes normally, else clause is executed
for i in range(5):
    print(i)
else:
    print("Loop completed normally")

# Loop is terminated by break, else clause is not executed
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("This won't be printed")
Output:
0
1
2
3
4
Loop completed normally
0
1
2

Nested Loops

You can place a loop inside another loop to create nested loops. The inner loop is executed once for each iteration of the outer loop.

Example: Nested Loops

# Print a multiplication table
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} * {j} = {i * j}")
    print("-----")
Output:
1 * 1 = 1
1 * 2 = 2
1 * 3 = 3
-----
2 * 1 = 2
2 * 2 = 4
2 * 3 = 6
-----
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9
-----

Loop Techniques

Python provides several built-in functions and techniques to make working with loops more efficient:

enumerate()

The enumerate() function adds a counter to an iterable, returning pairs of (index, item).

Example: enumerate()

fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
Output:
0: apple
1: banana
2: cherry

zip()

The zip() function combines multiple iterables into a single iterable of tuples.

Example: zip()

names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]

for name, age in zip(names, ages):
    print(f"{name} is {age} years old")
Output:
Alice is 25 years old
Bob is 30 years old
Charlie is 35 years old

List Comprehensions

List comprehensions provide a concise way to create lists based on existing lists or other iterables.

Example: List Comprehensions

# Create a list of squares
squares = [x**2 for x in range(1, 6)]
print(squares)

# Create a list of even numbers
evens = [x for x in range(10) if x % 2 == 0]
print(evens)

# Create a list of tuples
coordinates = [(x, y) for x in range(3) for y in range(2)]
print(coordinates)
Output:
[1, 4, 9, 16, 25]
[0, 2, 4, 6, 8]
[(0, 0), (0, 1), (1, 0), (1, 1), (2, 0), (2, 1)]
Note: List comprehensions are generally more concise and faster than equivalent for loops, but they can be harder to read if they become too complex. Use them judiciously.

Exception Handling

Exception handling allows you to gracefully handle errors that might occur during program execution.

try-except Statement

The try-except statement is used to catch and handle exceptions that might be raised in a block of code.

Example: Basic try-except

try:
    # Code that might raise an exception
    num = int(input("Enter a number: "))
    result = 10 / num
    print(f"10 / {num} = {result}")
except:
    # Code to handle the exception
    print("An error occurred")

Handling Specific Exceptions

You can specify which exceptions to catch and handle differently.

Example: Handling Specific Exceptions

try:
    num = int(input("Enter a number: "))
    result = 10 / num
    print(f"10 / {num} = {result}")
except ValueError:
    print("Invalid input. Please enter a valid number.")
except ZeroDivisionError:
    print("Cannot divide by zero.")
except:
    print("An unexpected error occurred.")

try-except-else-finally Statement

The try-except statement can have optional else and finally clauses:

  • The else clause is executed if no exception is raised in the try block.
  • The finally clause is always executed, whether an exception is raised or not.

Example: try-except-else-finally

try:
    num = int(input("Enter a number: "))
    result = 10 / num
except ValueError:
    print("Invalid input. Please enter a valid number.")
except ZeroDivisionError:
    print("Cannot divide by zero.")
else:
    # This block executes if no exception is raised
    print(f"10 / {num} = {result}")
finally:
    # This block always executes
    print("Execution completed")

Raising Exceptions

You can raise exceptions explicitly using the raise statement.

Example: Raising Exceptions

def validate_age(age):
    if age < 0:
        raise ValueError("Age cannot be negative")
    if age > 120:
        raise ValueError("Age is too high")
    return age

try:
    user_age = int(input("Enter your age: "))
    validated_age = validate_age(user_age)
    print(f"Your age is {validated_age}")
except ValueError as e:
    print(f"Error: {e}")

Creating Custom Exceptions

You can create your own exception classes by inheriting from the built-in Exception class or its subclasses.

Example: Custom Exceptions

class AgeError(Exception):
    """Exception raised for errors in the age input."""
    pass

def validate_age(age):
    if age < 0:
        raise AgeError("Age cannot be negative")
    if age > 120:
        raise AgeError("Age is too high")
    return age

try:
    user_age = int(input("Enter your age: "))
    validated_age = validate_age(user_age)
    print(f"Your age is {validated_age}")
except AgeError as e:
    print(f"Age Error: {e}")
except ValueError:
    print("Invalid input. Please enter a valid number.")

Context Managers (with Statement)

The with statement is used to wrap the execution of a block of code with methods defined by a context manager. It simplifies exception handling by encapsulating common patterns like acquiring and releasing resources.

Example: File Handling with Context Manager

# Without context manager
file = open("example.txt", "w")
try:
    file.write("Hello, World!")
finally:
    file.close()

# With context manager
with open("example.txt", "w") as file:
    file.write("Hello, World!")
# File is automatically closed when the block is exited
Note: The with statement ensures that resources are properly managed, even if an exception occurs. It's particularly useful for file operations, database connections, and network resources.

Conclusion

Control flow statements are essential for writing flexible and powerful Python programs. By mastering conditional statements, loops, and exception handling, you can create programs that make decisions, repeat actions, and gracefully handle errors.
