Indentation (Spacing) üëâ

Bhai, Python mein curly braces {} ka chakkar nahi hai! Yahan code blocks ko define karne ke liye indentation (yaani spaces) use hoti hai. Har line ke shuruat mein kitne spaces hain, wahi decide karta hai ki code kis block mein hai.

Semicolons Ka Chakkar ü§î

Bhai, C, Java ya JavaScript ki tarah Python mein statements ke end mein semicolons ; lagane ki zaroorat nahi hai! Lekin agar ek line mein multiple statements likhna ho toh semicolons use kar sakte ho.

Example: Semicolons in Python

# Semicolons ki zaroorat nahi

# Ek line mein multiple statements (par yeh style avoid karo)

üëâ Bro Tip: Semicolons use kar sakte ho, par Python mein ek line mein ek hi statement likhna best practice hai. Code clean aur readable rahega!

Basic Program Structure üèóÔ∏è

Python Scripts Kaise Likhte Hain üìú

Python script ek file hoti hai jisme Python code likha hota hai. Python files ka extension .py hota hai.

Example: Simple Python Script (hello.py)

# Yeh ek simple Python script hai

# Module import karna
import math

# Variable define karna
radius = 5

# Circle ka area calculate karna
area = math.pi * radius ** 2

# Result print karna
print(f"Radius {radius} wale circle ka area {area:.2f} hai")
Output:
Radius 5 wale circle ka area 78.54 hai
üí™ Bro Tip: Python mein f-strings (f"...") use karke variables ko strings ke andar easily add kar sakte ho. Yeh Python 3.6+ mein available hai aur bahut powerful feature hai!

Python Modules Kya Hote Hain üìö

Module ek Python file hoti hai jisme functions, classes aur variables define kiye hote hain. Doosre files mein import karke use kar sakte hain.

Example: Module Banana aur Use Karna

Pehle, mymodule.py naam ki file banao:

# mymodule.py
def greet(name):
    """Greeting message return karta hai"""
    return f"Hello, {name}!"

def add(a, b):
    """Do numbers ka sum return karta hai"""
    return a + b

PI = 3.14159

Phir, iss module ko doosri script mein use karo:

# main.py
import mymodule

# Module ke functions use karo
message = mymodule.greet("Rahul")
print(message)

result = mymodule.add(5, 3)
print(f"5 + 3 = {result}")

# Module ke variable access karo
print(f"PI ki value: {mymodule.PI}")
Output:
Hello, Rahul!
5 + 3 = 8
PI ki value: 3.14159

Python Packages Kya Hote Hain üì¶

Package ek directory hota hai jisme multiple module files aur ek special __init__.py file hoti hai. Yeh directory ko package ke taur par treat karne ke liye hota hai.

Classes Kaise Banate Hain üèóÔ∏è

Classes class keyword se define karte hain. Yeh data aur functionality ko ek saath bundle karne ka tarika hai.

Example: Class Definition and Usage

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"Namaste! Mera naam {self.name} hai aur main {self.age} saal ka hoon."

# Class ka instance create karna
person = Person("Rahul", 25)

# Instance ke method ko call karna
message = person.introduce()
print(message)
Output:
Namaste! Mera naam Rahul hai aur main 25 saal ka hoon.
üßê Bro Tip: __init__ method class ka constructor hota hai jo object create hote waqt automatically call hota hai. self parameter hamesha first parameter hota hai aur current instance ko refer karta hai.

Python Style Guide (PEP 8) üìè

PEP 8 Python code ke liye style guide hai. Isse follow karne se code readable aur consistent rehta hai.

PEP 8 Ke Main Rules

  • Har indentation level ke liye 4 spaces use karo
  • Lines ko 79 characters tak limit rakho
  • Functions, classes aur code ke bade blocks ko blank lines se separate karo
  • Operators ke around aur commas ke baad spaces use karo
  • Classes ke names CamelCase mein likho (jaise MyClass)
  • Functions aur variables ke names lowercase with underscores mein likho (jaise my_function, my_variable)
  • Constants ke names UPPERCASE with underscores mein likho (jaise MY_CONSTANT)

Example: PEP 8 Compliant Code

import os
import sys

# Constants
MAX_VALUE = 100
DEFAULT_NAME = "User"

class MyClass:
    """Yeh PEP 8 guidelines follow karne wali sample class hai."""
    
    def __init__(self, name=DEFAULT_NAME):
        self.name = name
    
    def say_hello(self):
        """Greeting message print karta hai."""
        print(f"Hello, {self.name}!")


def calculate_average(numbers):
    """Numbers ki list ka average calculate karta hai."""
    if not numbers:
        return 0
    
    total = sum(numbers)
    average = total / len(numbers)
    
    return average


if __name__ == "__main__":
    # MyClass ka instance create karo
    my_instance = MyClass("Rahul")
    my_instance.say_hello()
    
    # Kuch numbers ka average calculate karke print karo
    values = [10, 20, 30, 40, 50]
    avg = calculate_average(values)
    print(f"Average hai: {avg}")
üíØ Bro Tip: if __name__ == "__main__": block ka use karo jab code ko both as a script and as a module use karna ho. Isse code import karne par automatically execute nahi hoga, sirf direct run karne par hi execute hoga.
