Variables in Python ğŸ“¦

Python mein variable ek naam hai jo memory location ko represent karta hai. C++ ya Java ki tarah explicit declaration ki zaroorat nahi hai - bas value assign karo aur ho gaya!

Variable Names Ke Rules

  • Variable names mein letters, numbers, aur underscore (_) ho sakte hain
  • Variable name letter ya underscore se start hona chahiye
  • Number se start nahi kar sakte
  • Variable names case-sensitive hote hain (age, Age, aur AGE teen alag variables hain)
  • Python keywords use nahi kar sakte (jaise if, for, class, etc.)

Example: Sahi aur Galat Variable Names

# Sahi variable names
name = "Rahul"
age = 25
_id = 12345
user_name = "rahul_sharma"
userName = "rahul_sharma"
NAME = "RAHUL"
name2 = "Rahul Sharma"

# Galat variable names
2name = "Rahul"  # Number se start nahi kar sakte
user-name = "rahul_sharma"  # Hyphen allowed nahi hai
class = "Python"  # 'class' ek Python keyword hai

Dynamic Typing ğŸ”„

Python mein ek variable ka type runtime mein change ho sakta hai - yahi dynamic typing hai!

Example: Dynamic Typing

x = 10  # x ek integer hai
print(x, type(x))

x = "Hello"  # Ab x ek string ban gaya
print(x, type(x))

x = [1, 2, 3]  # Ab x ek list ban gaya
print(x, type(x))
Output:

Hello <class 'str'>
[1, 2, 3] <class 'list'>

Collection Data Types ğŸ“š

1. Lists ğŸ“‹

Lists ordered, mutable collections hain. Square brackets [] mein define kiye jate hain.

Example: Lists

# List banane ke tarike
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]
empty_list = []
nested_list = [1, [2, 3], 4]

print(fruits, type(fruits))
print(len(fruits))  # List length: 3
print(fruits[0])  # First element: "apple"
print(fruits[-1])  # Last element: "cherry"

# Lists mutable hain - change kar sakte hain
fruits[0] = "orange"
print(fruits)  # ["orange", "banana", "cherry"]

# List methods
fruits.append("mango")  # Add item at end
print(fruits)  # ["orange", "banana", "cherry", "mango"]

fruits.remove("cherry")  # Remove item
print(fruits)  # ["orange", "banana", "mango"]

popped = fruits.pop()  # Remove & return last item
print(popped, fruits)  # "mango" ["orange", "banana"]
ğŸ”¥ Bro Tip: Lists Python mein most versatile data structure hain! Jab bhi multiple items store karne hon aur order matter karta ho, list use karo.

2. Tuples ğŸ“Œ

Tuples lists ki tarah hain, lekin immutable (change nahi kar sakte). Parentheses () mein define kiye jate hain.

Example: Tuples

# Tuple banane ke tarike

ğŸ”’ Bro Tip: Tuples ka use tab karo jab data change nahi hona chahiye, jaise coordinates, database records, ya function se multiple values return karna ho.

3. Sets ğŸ¯

Sets unordered collections of unique items hain. Curly braces {} mein define kiye jate hain.

Example: Sets

# Set banane ke tarike
fruits = {"apple", "banana", "cherry"}
duplicates = {"apple", "banana", "apple", "cherry"}  # Duplicates automatically remove ho jate hain
empty_set = set()  # {} se empty dictionary banta hai, empty set nahi!

print(fruits, type(fruits))
print(duplicates)  # {"apple", "banana", "cherry"}

# Set operations
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a.union(b))  # {1, 2, 3, 4, 5, 6}
print(a.intersection(b))  # {3, 4}
print(a.difference(b))  # {1, 2}
ğŸ’ª Bro Tip: Sets ka best use duplicates remove karne ke liye hota hai. Aur membership testing (checking if item is in set) lists se much faster hota hai!

4. Dictionaries (dict) ğŸ“–

Dictionaries key-value pairs store karte hain. Curly braces {} mein key:value format mein define kiye jate hain.

# Dictionary methods
keys = person.keys()
values = person.values()

print(values)

# Iterate through dictionary
for key, value in person.items():
    print(key, value)

Type Conversion ğŸ”„

Python mein ek type se doosre type mein convert karna bahut easy hai:

Example: Type Conversion

# int() conversion
print(int(3.14))  # 3 (truncates, round nahi karta)
print(int("10"))  # 10
print(int(True))  # 1

# float() conversion
print(float(5))  # 5.0
print(float("3.14"))  # 3.14

# str() conversion
print(str(42))  # "42"
print(str(3.14))  # "3.14"

# bool() conversion
print(bool(1))  # True (non-zero numbers)
print(bool(0))  # False (zero)
print(bool("hello"))  # True (non-empty strings)
print(bool(""))  # False (empty string)
print(bool([]))  # False (empty collections)
