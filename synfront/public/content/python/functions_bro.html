Kya hai ye Functions? 🤔

Bro, functions matlab reusable code blocks! Baar baar same cheez likhne se better hai ek baar likh ke use karo jab chahiye. Samjhe?

Functions ke fayde:

  • Code reuse kar sakte ho - ek baar likho, hazaar baar use karo!
  • Complex problems ko chote-chote parts mein break kar sakte ho
  • Code organize rehta hai, bro - clean and sexy!

Function kaise banate hai? 💪

Bro, function banane ke liye def keyword use karte hai. Simple!

def mera_function():
    print("Aur bhai, kya haal hai?")

# Function ko call karne ke liye
mera_function()

Output:
Aur bhai, kya haal hai?

Parameters aur Arguments 🎯

Bro, function ko data pass karne ke liye parameters use karte hai:

def greeting(naam):
    print(f"Kya scene hai {naam} bhai?")

# Ab call karo with argument
greeting("Rahul")
greeting("Priya")

Output:
Kya scene hai Rahul bhai?
Kya scene hai Priya bhai?

Multiple Parameters 🔥

Ek se zyada parameters bhi de sakte hai, bro:

def add_numbers(x, y):
    return x + y

result = add_numbers(5, 3)
print(f"5 + 3 = {result}")

Output:
5 + 3 = 8

Default Parameters 😌

Bro, kabhi-kabhi argument nahi dena ho toh default value set kar sakte hai:

def greeting(naam="dost"):
    print(f"Kya scene hai {naam}?")

greeting()  # Default value use hoga
greeting("Vikram")

Output:
Kya scene hai dost?
Kya scene hai Vikram?

Return Statement 🔙

Function se value wapas lene ke liye return use karte hai:

def multiply(x, y):
    return x * y

result = multiply(4, 5)
print(f"4 * 5 = {result}")

Output:
4 * 5 = 20

Keyword Arguments 🔑

Parameter ka naam specify karke bhi arguments pass kar sakte hai:

def student_info(name, age, city):
    print(f"{name} is {age} years old from {city}")

# Order matter nahi karta with keyword arguments
student_info(city="Mumbai", name="Raj", age=21)

Output:
Raj is 21 years old from Mumbai

Variable Length Arguments (*args) 📏

Bro, jab pata nahi kitne arguments aayenge tab *args use karo:

def add_all(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(add_all(1, 2, 3))
print(add_all(5, 10, 15, 20))

Output:
6
50

Keyword Variable Length Arguments (**kwargs) 🔠

Aur jab key-value pairs pass karne ho tab **kwargs use karo:

def student_details(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

student_details(name="Rahul", age=20, city="Delhi", hobby="Gaming")

Output:
name: Rahul
age: 20
city: Delhi
hobby: Gaming

Lambda Functions (One-liners) 🚀

Bro, chote functions ke liye lambda use kar sakte hai - one-line mein kaam khatam!

# Normal function
def square(x):
    return x * x

# Same cheez lambda se
square_lambda = lambda x: x * x

print(square(5))
print(square_lambda(5))

Output:
25
25

Pro Tip: Bro, functions ko variables ki tarah pass kar sakte hai dusre functions mein! Ye advanced programming mein bahut useful hota hai.

Scope of Variables 🔍

Bro, variables ka scope important hai - local variables sirf function ke andar, global variables pure program mein:

global_var = "Main hoon global"

def test_scope():
    local_var = "Main hoon local"
    print(local_var)
    print(global_var)  # Global variable access kar sakte hai

test_scope()
print(global_var)
# print(local_var)  # Error aayega - local_var sirf function ke andar hai

Output:
Main hoon local
Main hoon global
Main hoon global

Recursion - Function khud ko call kare 🔄

Bro, recursion matlab function khud ko call karta hai - mind-blowing concept hai!

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)

print(f"5! = {factorial(5)}")

Output:
5! = 120

Yaad rakhna: Recursion mein base case zaroor define karo, warna infinite loop mein fas jaoge!
