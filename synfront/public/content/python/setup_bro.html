Python Install Karna üîß

Windows Pe Install Karna

1 Python Installer Download Karo:

Python ki official website pe jao aur latest version download karo Windows ke liye.

2 Installer Run Karo:

Downloaded file ko open karo aur dhyan se "Add Python to PATH" wale box ko check karo, phir "Install Now" pe click karo. Agar yeh step miss kiya toh baad mein bahut pareshani hogi, so dhyan se!

3 Installation Check Karo:

Command Prompt kholo aur yeh command type karo:

python --version

Agar sab sahi hai toh Python ka version number dikhega. Agar error aaye toh installation mein kuch gadbad hai!

macOS Pe Install Karna

1 Homebrew Se Install Karo (Recommended):

Agar tumhare paas Homebrew installed hai, toh bas yeh command run karo:

brew install python

Itna simple! üç∫

2 Official Installer Se Install Karo:

Ya phir, Python website se macOS installer download karke install kar sakte ho.

3 Installation Check Karo:

Terminal kholo aur run karo:

python3 --version

Linux Pe Install Karna

1 Package Manager Se Install Karo:

Bhai Linux mein toh mostly Python pre-installed aata hai. Agar nahi hai toh package manager se install kar sakte ho.

Ubuntu/Debian ke liye:

sudo apt update
sudo apt install python3 python3-pip

Fedora ke liye:

sudo dnf install python3 python3-pip
2 Installation Check Karo:

Terminal kholo aur run karo:

python3 --version

Code Editor Setup Karna üíª

Bhai, Python code toh kisi bhi text editor mein likh sakte ho, lekin ek achha code editor ya IDE use karne se productivity next level pe pahunch jayegi. Yeh rahi kuch best options:

Visual Studio Code (Beginners Ke Liye Best Hai)

1 VS Code Download Karo:

VS Code website se apne system ke liye version download karo.

2 Python Extension Install Karo:

VS Code kholo, Extensions icon pe click karo (left side mein) aur "Python" search karo. Microsoft wala Python extension install karo. Bas itna sa kaam hai!

üî• Bro Tip: VS Code mein "Python: Select Interpreter" command run karke apne system ke Python ko select karo. Command palette open karne ke liye Ctrl+Shift+P (Windows/Linux) ya Cmd+Shift+P (Mac) press karo.

PyCharm (Pro Developers Ke Liye)

PyCharm ek full-featured IDE hai jo specifically Python development ke liye banaya gaya hai.

1 PyCharm Download Karo:

JetBrains website se Community (free) ya Professional edition download karo.

2 Install Karke Setup Karo:

Installation instructions follow karo aur PyCharm first time open karne pe new Python project create karo.

üí™ Bro Tip: PyCharm thoda heavy hai but features ka baap hai! Agar serious Python development karna hai toh isse better kuch nahi.

Virtual Environments Banana üß™

Bhai virtual environments bahut important cheez hai! Yeh alag-alag projects ke liye isolated environments create karta hai, jisse dependencies ka jhanjhat nahi hota.

venv Use Karna (Built-in Hai)

1 Virtual Environment Create Karo:

Apne project directory mein jao aur yeh command run karo:

# Windows
python -m venv venv

# macOS/Linux
python3 -m venv venv

Yeh 'venv' naam ka folder create karega jisme saara environment ka saman hoga.

2 Virtual Environment Activate Karo:

Windows ke liye:

venv\Scripts\activate

macOS/Linux ke liye:

source venv/bin/activate

Activate hone pe command prompt/terminal mein (venv) dikhai dega.

3 Packages Install Karo:

Virtual environment activate hone ke baad, pip se packages install kar sakte ho:

pip install package_name
üöÄ Bro Tip: Har project ke liye alag virtual environment banao! Isse packages ka version conflict nahi hoga aur project clean rahega.

Anaconda/Miniconda Use Karna

Anaconda data science aur machine learning ke liye bahut popular hai. Isme saare important packages pre-installed aate hain.

1 Anaconda Download Karke Install Karo:

Anaconda website se apne system ke liye version download karo.

2 Conda Environment Create Karo: 
conda create --name myenv python=3.9

Yeh 'myenv' naam ka environment create karega Python 3.9 ke saath.

3 Environment Activate Karo: 
# Windows/macOS/Linux
conda activate myenv
üß† Bro Tip: Agar data science, machine learning ya scientific computing karna hai toh Anaconda best option hai! Saare important packages ek saath mil jayenge.

First Python Program Likhna üéâ

Ab time aa gaya hai apna pehla Python program likhne ka! Chalo ek simple "Hello, World!" program se shuruat karte hain.

1 New File Create Karo:

Apne code editor mein ek new file create karo aur use hello.py naam se save karo.

2 Code Likho:

File mein yeh line likho:

print("Hello, World! Mera first Python program!")
3 Program Run Karo:

Terminal ya command prompt mein, apne file wale directory mein jao aur run karo:

# Windows
python hello.py

# macOS/Linux
python3 hello.py

Terminal mein "Hello, World! Mera first Python program!" print ho jayega. Congrats! üéä

üòé Bro Tip: VS Code mein Python file open karke right-click karke "Run Python File in Terminal" option se bhi run kar sakte ho. Ya phir top-right corner mein play button bhi hota hai!

pip Se Packages Install Karna üì¶

pip Python ka package installer hai. Isse PyPI (Python Package Index) aur doosre repositories se packages install kar sakte ho.

Basic pip Commands

  • Package Install Karna: 
    pip install package_name
  • Specific Version Install Karna: 
    pip install package_name==1.0.0
  • Package Upgrade Karna: 
    pip install --upgrade package_name
  • Package Uninstall Karna: 
    pip uninstall package_name
  • Installed Packages List Dekhna: 
    pip list
  • Requirements File Generate Karna: 
    pip freeze > requirements.txt
  • Requirements File Se Install Karna: 
    pip install -r requirements.txt
üíØ Bro Tip: Project ke liye requirements.txt file zaroor banao! Isse doosre log bhi aasani se tumhara project run kar payenge. Teamwork mein yeh bahut important hai!
