File I/O Kya Hai? ü§î

Bro, File I/O matlab File Input/Output - basically computer ke files ke saath kaam karna! Python mein ye bahut easy hai, aur isse aap data save kar sakte ho, read kar sakte ho, aur process kar sakte ho.

File I/O Kyu Important Hai?

  • Data permanent store karne ke liye
  • Bade datasets process karne ke liye
  • Configuration settings save karne ke liye
  • Log files banane ke liye
  • Different programs ke beech data exchange karne ke liye

Files Kaise Open aur Close Karte Hai? üîì

Bro, files open karne ke liye open() function use karte hai:

# Basic syntax
file = open('filename.txt', 'mode')
# File operations
file.close()  # Always close files when done!

# Pro wala approach - with statement use karo
with open('filename.txt', 'mode') as file:
    # File operations
    pass  # Block khatam, file automatically close!

File Opening Modes:

  • 'r': Read mode (default) - file ko sirf padhne ke liye
  • 'w': Write mode - file mein likhne ke liye (purana content delete ho jayega!)
  • 'a': Append mode - file ke end mein add karne ke liye
  • 'x': Exclusive creation - new file banayega, agar file already hai to error
  • 'b': Binary mode - images, audio jaise binary files ke liye (e.g., 'rb', 'wb')
  • 't': Text mode (default) - text files ke liye
  • '+': Update mode - reading aur writing dono ke liye

Pro Tip: Bro, hamesha with statement use karo! Isse file automatically close ho jati hai, even if koi error aaye. No tension!

Files Se Data Kaise Read Karte Hai? üìñ

Bro, files se data read karne ke kai tarike hai:

Puri File Ek Baar Mein Read Karna:

# Puri file ko ek string ki tarah read karo
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Puri file ko lines ki list ki tarah read karo
with open('example.txt', 'r') as file:
    lines = file.readlines()
    for line in lines:
        print(line.strip())  # strip() newline character hatata hai

Line by Line Read Karna:

# Method 1: readline() use karke
with open('example.txt', 'r') as file:
    line = file.readline()
    while line:
        print(line.strip())
        line = file.readline()

# Method 2: File object pe loop chalake (ye zyada efficient hai)
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

Specific Amount of Data Read Karna:

with open('example.txt', 'r') as file:
    # First 10 characters read karo
    chunk = file.read(10)
    print(chunk)

Files Mein Data Kaise Write Karte Hai? ‚úçÔ∏è

Bro, files mein data write karne ke liye:

Text Write Karna:

# File mein string write karo (new file create hogi ya existing overwrite hogi)
with open('output.txt', 'w') as file:
    file.write('Kya scene hai bro!\n')
    file.write('Ye new line hai.')

# Existing file mein append karo
with open('output.txt', 'a') as file:
    file.write('\nYe line append hui hai.')

Multiple Lines Write Karna:

lines = ['First line', 'Second line', 'Third line']

# Method 1: write() use karke
with open('output.txt', 'w') as file:
    for line in lines:
        file.write(line + '\n')

# Method 2: writelines() use karke
with open('output.txt', 'w') as file:
    # Har string ke end mein newline add karo
    file.writelines(line + '\n' for line in lines)

File Positions Control Karna üéÆ

Bro, file mein cursor ki position control karne ke liye seek() aur tell() methods use karte hai:

with open('example.txt', 'r') as file:
    # Current position check karo
    position = file.tell()
    print(f"Current position: {position}")
    
    # First 5 characters read karo
    content = file.read(5)
    print(f"Read: {content}")
    
    # New position check karo
    position = file.tell()
    print(f"New position: {position}")
    
    # File ke beginning mein jao
    file.seek(0)
    print(f"After seek(0): {file.tell()}")
    
    # File ke 10th byte pe jao
    file.seek(10)
    print(f"After seek(10): {file.tell()}")

Binary Files Ke Saath Kaam Karna üñºÔ∏è

Bro, binary files matlab non-text data jaise images, audio, ya programs. Inke saath kaam karne ke liye binary mode ('b') use karo:

# Binary file read karna (e.g., image)
with open('image.jpg', 'rb') as file:
    binary_data = file.read()
    print(f"File size: {len(binary_data)} bytes")

# Binary file mein write karna
with open('copy.jpg', 'wb') as file:
    file.write(binary_data)

File aur Directory Management üìÅ

Bro, Python ke os aur shutil modules se file aur directory operations kar sakte ho:

import os
import shutil

# Check karo ki file exist karti hai ya nahi
if os.path.exists('example.txt'):
    print('File exist karti hai')

# File information nikalo
if os.path.isfile('example.txt'):
    size = os.path.getsize('example.txt')
    print(f"File size: {size} bytes")
    print(f"Last modified: {os.path.getmtime('example.txt')}")

# Directory create karo
os.makedirs('new_directory', exist_ok=True)

# Directory mein files list karo
files = os.listdir('.')
print(files)

# File rename karo
os.rename('old_name.txt', 'new_name.txt')

# File delete karo
os.remove('file_to_delete.txt')

# File copy karo
shutil.copy('source.txt', 'destination.txt')

# File move karo
shutil.move('source.txt', 'new_location/source.txt')

# Directory aur uske contents delete karo
shutil.rmtree('directory_to_delete')

CSV Files Ke Saath Kaam Karna üìä

Bro, CSV (Comma-Separated Values) files data store karne ka common format hai. Python ka csv module isse handle karne mein help karta hai:

import csv

# CSV file read karna
with open('data.csv', 'r', newline='') as file:
    csv_reader = csv.reader(file)
    # Header row skip karo
    header = next(csv_reader)
    print(f"Headers: {header}")
    
    # Data rows read karo
    for row in csv_reader:
        print(row)

# CSV file mein write karna
data = [
    ['Name', 'Age', 'City'],
    ['Rahul', 25, 'Delhi'],
    ['Priya', 23, 'Mumbai'],
    ['Amit', 27, 'Bangalore']
]

with open('output.csv', 'w', newline='') as file:
    csv_writer = csv.writer(file)
    # Multiple rows ek saath write karo
    csv_writer.writerows(data)

JSON Files Ke Saath Kaam Karna üìù

Bro, JSON (JavaScript Object Notation) data exchange ka lightweight format hai. Python ka json module isse handle karta hai:

import json

# Python dictionary
data = {
    'name': 'Rahul',
    'age': 25,
    'city': 'Mumbai',
    'skills': ['Python', 'JavaScript', 'React'],
    'is_student': False
}

# JSON file mein write karna
with open('data.json', 'w') as file:
    json.dump(data, file, indent=4)  # indent for pretty formatting

# JSON file se read karna
with open('data.json', 'r') as file:
    loaded_data = json.load(file)
    print(loaded_data)

Custom File Handlers Banana üõ†Ô∏è

Bro, apne custom file handlers bhi bana sakte ho context managers use karke:

class MyFileHandler:
    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
        self.file = None
    
    def __enter__(self):
        self.file = open(self.filename, self.mode)
        return self.file
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        if self.file:
            self.file.close()
        # Return False to propagate exceptions, True to suppress them
        return False

# Custom context manager use karna
with MyFileHandler('example.txt', 'r') as file:
    content = file.read()
    print(content)
