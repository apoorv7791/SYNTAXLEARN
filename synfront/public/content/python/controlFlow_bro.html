Python Control Flow Samajho üö¶

Bhai, control flow matlab program mein code ka execution flow control karna. Isse hum decide kar sakte hain ki konsa code kab execute hoga. Python mein control flow ke liye conditionals (if-else), loops (for, while), aur exception handling jaisi cheezein use hoti hain. Chalo inhe detail mein samajhte hain!

Conditional Statements üîÄ

Conditional statements se hum code ko conditions ke basis par execute kar sakte hain. Matlab, agar condition true hai to kuch code chalao, warna kuch aur.

if Statement

if statement ka use tab karte hain jab humein koi code sirf tab execute karna ho jab condition true ho.

Example: if Statement

umar = 20

if umar >= 18:
    print("Aap adult ho.")
    print("Aap vote de sakte ho.")
Output:
Aap adult ho.
Aap vote de sakte ho.
üí° Bro Tip: Python mein indentation (spaces) bahut important hai! Har if statement ke baad colon : lagao aur uske baad wale code ko indent karo (4 spaces). Indentation se hi Python samajhta hai ki konsa code if block ke andar hai.

if-else Statement

if-else statement se hum do options create kar sakte hain - agar condition true hai to kuch karo, warna kuch aur karo.

Example: if-else Statement

umar = 15

if umar >= 18:
    print("Aap adult ho.")
    print("Aap vote de sakte ho.")
else:
    print("Aap minor ho.")
    print("Aap abhi vote nahi de sakte.")
Output:
Aap minor ho.
Aap abhi vote nahi de sakte.

if-elif-else Statement

if-elif-else statement se hum multiple conditions check kar sakte hain ek ke baad ek.

Example: if-elif-else Statement

marks = 75

if marks >= 90:
    grade = 'A'
elif marks >= 80:
    grade = 'B'
elif marks >= 70:
    grade = 'C'
elif marks >= 60:
    grade = 'D'
else:
    grade = 'F'

print(f"Aapka grade {grade} hai")
Output:
Aapka grade C hai
üî• Bro Tip: elif ka matlab "else if" hota hai. Jitne chahe utne elif use kar sakte ho. Python conditions ko upar se neeche check karta hai aur jaise hi koi condition true milti hai, wahi block execute hota hai aur baaki skip ho jate hain.

Nested if Statements

Ek if statement ke andar doosra if statement bhi likh sakte hain. Ise nested if statements kehte hain.

Example: Nested if Statements

umar = 25
license_hai = True

if umar >= 18:
    print("Aap driving ke liye old enough ho.")
    if license_hai:
        print("Aap legally drive kar sakte ho.")
    else:
        print("Pehle license banwao bhai.")
else:
    print("Aap driving ke liye too young ho.")
Output:
Aap driving ke liye old enough ho.
Aap legally drive kar sakte ho.

Ternary Operator (One-liner if-else)

Python mein ek line mein if-else likhne ka shortcut hai, jise ternary operator kehte hain.

Example: Ternary Operator

umar = 20
status = "adult" if umar >= 18 else "minor"
print(status)

# Yeh same hai:
# if umar >= 18:
#     status = "adult"
# else:
#     status = "minor"
Output:
adult
üëå Bro Tip: Ternary operator simple conditions ke liye best hai. Complex logic ke liye normal if-else use karo, warna code confusing ho jayega.

Loops (Repetition) üîÑ

Loops ka use code ko baar-baar execute karne ke liye karte hain. Python mein do main types ke loops hain: for loops aur while loops.

for Loop

for loop ka use kisi sequence (list, tuple, string, range) ke har item par iterate karne ke liye karte hain.

Example: for Loop with a List

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
Output:
apple
banana
cherry

Example: for Loop with range()

# range(5) generates numbers from 0 to 4
for i in range(5):
    print(i)

# range(start, stop) generates numbers from start to stop-1
for i in range(2, 6):
    print(i)

# range(start, stop, step) generates numbers with step
for i in range(1, 10, 2):  # 1 se start, 10 se pehle tak, har 2 number par
    print(i)
Output:
0
1
2
3
4
2
3
4
5
1
3
5
7
9
üî¢ Bro Tip: range() function bahut useful hai loops mein! Yaad rakho, range(n) 0 se n-1 tak numbers generate karta hai, n include nahi hota.

while Loop

while loop tab tak code execute karta hai jab tak condition true hai.

Example: while Loop

count = 0

while count < 5:
    print(count)
    count += 1  # Increment karna mat bhoolna!
Output:
0
1
2
3
4
‚ö†Ô∏è Warning: While loop mein condition update karna kabhi mat bhoolna! Agar condition kabhi false nahi hogi to loop hamesha chalte rahega (infinite loop) aur program hang ho jayega.

Loop Control Statements

Python mein loops ko control karne ke liye kuch special statements hain:

break Statement

break statement loop ko turant terminate kar deta hai aur control loop ke bahar wale code par chala jata hai.

Example: break Statement

for i in range(10):
    if i == 5:
        break  # Loop se bahar nikal jao jab i = 5 ho
    print(i)
Output:
0
1
2
3
4
üö™ Bro Tip: break ka use tab karo jab aapko specific condition milne par loop se turant bahar nikalna ho. Jaise ki search karte time jab item mil jaye.

continue Statement

continue statement current iteration ko skip karke loop ke next iteration par chala jata hai.

Example: continue Statement

for i in range(10):
    if i % 2 == 0:  # Even numbers ko skip karo
        continue
    print(i)
Output:
1
3
5
7
9
‚è≠Ô∏è Bro Tip: continue ka use tab karo jab aapko kuch specific items ko process nahi karna ho, lekin loop chalate rehna ho.

else Clause in Loops

Python mein loops ke saath else clause bhi use kar sakte hain, jo tab execute hota hai jab loop normally complete ho jaye (break se nahi).

Example: else Clause in for Loop

# Loop normally complete hota hai, else execute hota hai
for i in range(5):
    print(i)
else:
    print("Loop normally complete hua")

# Loop break se terminate hota hai, else execute nahi hota
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Yeh print nahi hoga")
Output:
0
1
2
3
4
Loop normally complete hua
0
1
2
ü§î Bro Tip: Loop ke saath else thoda confusing lag sakta hai, lekin yeh useful hai jab aapko pata karna ho ki loop normally complete hua ya break se terminate hua.

Nested Loops

Ek loop ke andar doosra loop likh sakte hain. Ise nested loops kehte hain. Inner loop outer loop ke har iteration par poora execute hota hai.

Example: Nested Loops

# Multiplication table print karna

Pro Loop Techniques

Python mein loops ko aur powerful banane ke liye kuch special functions hain:

enumerate()

enumerate() function iterable ke items ke saath unka index bhi provide karta hai.

Example: enumerate()

fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
Output:
0: apple
1: banana
2: cherry

zip()

zip() function multiple lists ko parallel iterate karne ke liye use hota hai.

Example: zip()

names = ["Rahul", "Priya", "Amit"]
ages = [25, 22, 28]

for name, age in zip(names, ages):
    print(f"{name} ki age {age} hai")
Output:
Rahul ki age 25 hai
Priya ki age 22 hai
Amit ki age 28 hai
üîó Bro Tip: zip() function related data ko combine karne ke liye perfect hai. Lekin dhyan rahe, agar lists ki length different hai to shortest list ke hisaab se zip hoga.

List Comprehensions

List comprehensions ek line mein list create karne ka shortcut hai.

Example: List Comprehensions

# Squares ki list banao
squares = [x**2 for x in range(1, 6)]
print(squares)

# Even numbers ki list banao
evens = [x for x in range(10) if x % 2 == 0]
print(evens)

# Nested list comprehension
matrix = [[i*j for j in range(1, 4)] for i in range(1, 4)]
print(matrix)
Output:
[1, 4, 9, 16, 25]
[0, 2, 4, 6, 8]
[[1, 2, 3], [2, 4, 6], [3, 6, 9]]
üöÄ Bro Tip: List comprehensions normal for loops se faster hote hain aur code bhi compact ho jata hai. Lekin zyada complex logic ke liye normal for loop hi use karo, warna code readable nahi rahega.

Exception Handling üõ°Ô∏è

Exception handling se hum errors ko gracefully handle kar sakte hain, taki program crash na ho.

try-except Statement

try-except statement ka use potential errors wale code ko handle karne ke liye karte hain.

Example: Basic try-except

try:
    # Code jisme error ho sakta hai
    num = int(input("Ek number daalo: "))
    result = 10 / num
    print(f"10 / {num} = {result}")
except:
    # Error handle karne ke liye code
    print("Kuch gadbad ho gayi!")

Specific Exceptions Handle Karna

Alag-alag types ke errors ko alag tarah se handle kar sakte hain.

Example: Handling Specific Exceptions

try:
    num = int(input("Ek number daalo: "))
    result = 10 / num
    print(f"10 / {num} = {result}")
except ValueError:
    print("Bhai number daal, text nahi!")
except ZeroDivisionError:
    print("Zero se divide nahi kar sakte!")
except:
    print("Koi aur error ho gaya!")
üõ°Ô∏è Bro Tip: Specific exceptions handle karna better hai than general except block use karna. Isse aapko pata chalta hai ki exactly kya error hua aur usko specifically handle kar sakte ho.

try-except-else-finally Statement

try-except ke saath optional else aur finally clauses bhi use kar sakte hain:

  • else tab execute hota hai jab try block mein koi error nahi hota.
  • finally hamesha execute hota hai, chahe error ho ya na ho.

Example: try-except-else-finally

try:
    num = int(input("Ek number daalo: "))
    result = 10 / num
except ValueError:
    print("Bhai number daal, text nahi!")
except ZeroDivisionError:
    print("Zero se divide nahi kar sakte!")
else:
    # Yeh tab execute hota hai jab koi error nahi hota
    print(f"10 / {num} = {result}")
finally:
    # Yeh hamesha execute hota hai
    print("Program khatam!")
üßπ Bro Tip: finally block ka use resources clean up karne ke liye karo, jaise files close karna ya database connections band karna. Yeh guarantee deta hai ki cleanup code hamesha run hoga, chahe error ho ya na ho.

Raising Exceptions

Khud se bhi exceptions raise kar sakte hain raise statement se.

Example: Raising Exceptions

def validate_age(age):
    if age < 0:
        raise ValueError("Age negative nahi ho sakti")
    if age > 120:
        raise ValueError("Itni badi age? Jhoot mat bolo!")
    return age

try:
    user_age = int(input("Apni age batao: "))
    validated_age = validate_age(user_age)
    print(f"Aapki age {validated_age} hai")
except ValueError as e:
    print(f"Error: {e}")
üö® Bro Tip: raise ka use tab karo jab aapko pata ho ki current situation mein program aage nahi badh sakta. User ko clear error message do taki woh samajh sake ki kya problem hai.

with Statement (Context Managers) üîí

with statement ka use resources (jaise files) ko automatically manage karne ke liye karte hain, taki unhe manually close karna na pade.

Example: File Handling with Context Manager

# Without context manager
file = open("example.txt", "w")
try:
    file.write("Hello, World!")
finally:
    file.close()

# With context manager (much better!)
with open("example.txt", "w") as file:
    file.write("Hello, World!")
# File automatically close ho jayegi
üîê Bro Tip: Hamesha with statement use karo file operations ke liye. Isse file automatically close ho jayegi, chahe code normally execute ho ya error aaye. Resource leaks se bachne ka best tarika hai yeh!
