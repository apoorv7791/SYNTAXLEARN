Python Modules and Packages - Bro Mode 😎

Modules Kya Hote Hai? 🤔

Bro, modules matlab pre-written code files jo aap apne program mein use kar sakte ho. Baar baar same code likhne se better hai, use a module!

Modules Kyu Use Kare?

  • Code organize rehta hai - clean and sexy!
  • Ek baar likho, hazaar baar use karo
  • Apna code dusron se share kar sakte ho
  • Bade projects ko manage karna easy ho jata hai

Modules Kaise Import Karte Hai? 📥

Bro, modules import karne ke different tarike hai:

# Pura module import karo
import math

# Module use karo
radius = 5
area = math.pi * radius**2
print(f"Circle ka area: {area}")

# Specific cheezein import karo
from math import pi, sqrt

area = pi * radius**2
print(f"Circle ka area: {area}")
print(f"16 ka square root: {sqrt(16)}")

# Alias ke saath import karo
import math as m

print(f"Pi: {m.pi}")

# Sab kuch import karo (not recommended)
from math import *

print(f"Pi: {pi}")

Output:
Circle ka area: 78.53981633974483
Circle ka area: 78.53981633974483
16 ka square root: 4.0
Pi: 3.141592653589793
Pi: 3.141592653589793

Pro Tip: Bro, from module import * se bachna chahiye. Isse code confusing ho jata hai aur functions kahan se aa rahe hai, pata nahi chalta!

Apna Module Kaise Banaye? 🛠️

Bro, apna module banane ke liye bas ek Python file create karo with functions, classes, ya variables:

# File: mymath.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

PI = 3.14159

# Dusre file mein
import mymath

print(mymath.add(5, 3))
print(mymath.subtract(10, 4))
print(mymath.PI)

Output:
8
6
3.14159

Module Kahan Se Load Hote Hai? 🔍

Bro, Python modules ko in jagahon se dhundhta hai:

  1. Current directory jahan se script run ho raha hai
  2. PYTHONPATH environment variable mein listed directories
  3. Default installation directories

Search path check karne ke liye:

import sys
print(sys.path)

if __name__ == "__main__" Kya Hai? 🧐

Bro, ye trick hai jisse aap code likh sakte ho jo both as a module import ho sake aur as a script run bhi ho sake:

# File: mymath.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

PI = 3.14159

# Ye code sirf tab chalega jab file directly run ho
if __name__ == "__main__":
    print("mymath module testing kar rahe hai")
    print(f"5 + 3 = {add(5, 3)}")
    print(f"10 - 4 = {subtract(10, 4)}")
    print(f"PI = {PI}")

Output (jab script ke tarah run karte hai):
mymath module testing kar rahe hai
5 + 3 = 8
10 - 4 = 6
PI = 3.14159

Packages Kya Hote Hai? 📦

Bro, package matlab modules ka folder with a special __init__.py file. Isse aap related modules ko organize kar sakte ho hierarchy mein.

mypackage/ ├── __init__.py ├── module1.py ├── module2.py └── subpackage/ ├── __init__.py └── module3.py

__init__.py File Kya Hai?

Bro, __init__.py file empty bhi ho sakti hai, but ye zaroori hai directory ko package banane ke liye. Isse package initialize bhi kar sakte hai.

Packages Se Import Kaise Karte Hai? 🧩

# Package se module import karo
import mypackage.module1

# Package ke module se specific function import karo
from mypackage.module1 import some_function

# Subpackage se module import karo
import mypackage.subpackage.module3

# Alias ke saath import karo
import mypackage.module2 as m2

Package Kaise Banate Hai? 🏗️

Bro, simple package structure banane ke liye:

# File: mypackage/__init__.py
print("mypackage initialize ho raha hai")

# File: mypackage/calculations.py
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

# File: mypackage/conversions.py
def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

def fahrenheit_to_celsius(fahrenheit):
    return (fahrenheit - 32) * 5/9

# Dusre file mein use karne ke liye
from mypackage import calculations, conversions

print(calculations.add(5, 3))
print(calculations.multiply(4, 2))
print(conversions.celsius_to_fahrenheit(25))

Output:
mypackage initialize ho raha hai
8
8
77.0

__all__ Se Export Control Karna 🎮

Bro, __all__ variable se aap control kar sakte ho ki from package import * karne par kya import hoga:

# File: mypackage/__init__.py
__all__ = ['calculations']  # Sirf calculations module export hoga

# File: mypackage/calculations.py
__all__ = ['add']  # Sirf add function export hoga

def add(a, b):
    return a + b

def multiply(a, b):  # Ye * se import nahi hoga
    return a * b

Standard Library Modules - Free Maal! 🎁

Bro, Python ke saath bahut saare ready-made modules aate hai:

  • math: Mathematical functions ke liye
  • random: Random numbers generate karne ke liye
  • datetime: Date aur time handle karne ke liye
  • os: Operating system ke saath interact karne ke liye
  • sys: System-specific parameters aur functions
  • json: JSON data handle karne ke liye
  • re: Regular expressions ke liye
  • collections: Special data containers

Examples:

# Math module
import math
print(math.sqrt(16))

# Random module
import random
print(random.randint(1, 10))

# Datetime module
import datetime
print(datetime.datetime.now())

# OS module
import os
print(os.getcwd())

# JSON module
import json
data = {'name': 'Rahul', 'age': 25}
json_string = json.dumps(data)
print(json_string)
print(json.loads(json_string))

Third-Party Packages - Extra Power! 💪

Bro, Python ecosystem mein hazaron third-party packages hai. Inhe pip se install kar sakte ho:

# Package install karo
pip install requests

# Package use karo
import requests
response = requests.get('https://api.github.com')
print(response.status_code)
print(response.json())

Virtual Environments - Alag Duniya! 🌍

Bro, virtual environments se aap alag-alag projects ke liye isolated environments create kar sakte ho:

# Virtual environment create karo
python -m venv myenv

# Virtual environment activate karo
# Windows pe:
myenv\Scripts\activate
# Unix ya MacOS pe:
source myenv/bin/activate

# Virtual environment mein packages install karo
pip install requests

# Kaam khatam, deactivate karo
deactivate

Pro Tip: Bro, har project ke liye virtual environment use karo! Isse different projects ke dependencies mein conflict nahi hoga aur life easy rahegi!
